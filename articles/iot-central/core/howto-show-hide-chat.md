# <a name="toggle-live-chat"></a>Live chat in-/uitschakelen

Dit how-to-artikel laat zien hoe je de live chat schakelen in je IoT Central-toepassing. U live chat gebruiken om toegang te krijgen tot technische ondersteuning.

> [!NOTE]
> De chatoptie is alleen beschikbaar voor toepassingen die zijn gemaakt met behulp van het gratis prijsplan.

## <a name="chat-with-us"></a>Chat met ons

Open uw IoT Central-toepassing en selecteer **Chat met ons**om technische ondersteuning te krijgen.

![Chat met ons](media/howto-show-hide-chat/chat-with-us.png)

U een vraag typen zoals weergegeven in de volgende schermafbeelding:

![Formulier Apparaatgegevens configureren met details voor instellingen en eigenschappen](media/howto-show-hide-chat/sample-chat.png)

## <a name="hide-chat"></a>Chat verbergen

Als u de chat wilt verbergen, kiest **u Chat verbergen** in het **Help-deelvenster:**

 ![Chat verbergen](media/howto-show-hide-chat/hide-chat.png)

## <a name="enable-chat"></a>Chat inschakelen

Als u de chat wilt weergeven, kiest **u Chat weergeven** in het **Help-deelvenster:**

 ![Chat weergeven](media/howto-show-hide-chat/show-chat.png)
