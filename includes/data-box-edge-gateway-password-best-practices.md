Houd rekening met deze aanbevolen procedures:

- We raden u aan alle wachtwoorden op een veilige plaats op te slaan, zodat u een wachtwoord niet opnieuw hoeft in te stellen als het wordt vergeten. De beheerservice kan bestaande wachtwoorden niet ophalen. Het kan ze alleen resetten via de Azure-portal. Als u een wachtwoord opnieuw instelt, moet u alle gebruikers hiervan op de hoogte stellen voordat u het opnieuw instelt.
- U hebt op afstand toegang tot de Windows PowerShell-interface van uw apparaat via HTTP. Als best practice voor beveiliging moet u HTTP alleen gebruiken op vertrouwde netwerken.
- Zorg ervoor dat apparaatwachtwoorden sterk en goed beschermd zijn. Volg de [best practices voor wachtwoorden.](https://docs.microsoft.com/azure/security/fundamentals/identity-management-best-practices#enable-password-management)
