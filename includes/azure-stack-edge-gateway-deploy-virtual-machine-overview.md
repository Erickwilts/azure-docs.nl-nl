U kunt virtuele machines (VM's) maken en beheren op een Azure Stack Edge-apparaat met behulp van API's. Deze API's zijn standaard Azure Resource Manager-API's die worden aangeroepen met behulp van het lokale Azure Stack Edge-eindpunt. De Azure Resource Manager-API's bieden een consistente managementlaag, waarmee u in dit geval VM's kunt maken, bijwerken en verwijderen in een lokaal abonnement dat bestaat op het apparaat. U kunt verbinding maken met de versie van Azure Resource Manager die wordt uitgevoerd op Azure Stack Edge via Azure PowerShell-cmdlets.
