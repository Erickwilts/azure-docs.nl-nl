# <a name="enabling-azure-active-directory-identity-protection"></a>Azure Active Directory Identity Protection inschakelen

Azure Active Directory Identity Protection is een mogelijkheid van Azure Active Directory (Azure AD). Met Azure AD Identity Protection kunt u het volgende doen:

- Een geconsolideerde weer gave van gemarkeerde gebruikers en risico gebeurtenissen die zijn gedetecteerd met machine learning algoritmen ophalen
- Op risico gebaseerd beleid voor voorwaardelijke toegang instellen om uw gebruikers automatisch te beveiligen
- Beveiligings postuur verbeteren door te handelen op beveiligings problemen 

Zie [Azure Active Directory Identity Protection](../active-directory-identityprotection.md)voor meer informatie.

In dit artikel wordt beschreven hoe u Azure Active Directory Identity Protection inschakelt.

## <a name="steps-to-enable-azure-active-directory-identity-protection"></a>Stappen voor het inschakelen van Azure Active Directory Identity Protection

1. Meld u aan bij uw [Azure Portal](https://ms.portal.azure.com/) als globale beheerder. 

2. Klik in de Azure Portal op **Marketplace**.
   
    ![Maken](./media/enable/01.png "Maken")

3. Klik in de lijst toepassingen op **identiteit**.
   
    ![Maken](./media/enable/05.png "Maken")

4. Klik op **Azure AD Identity Protection**.
   
    ![Maken](./media/enable/06.png "Maken")

5. Klik op de Blade **Azure AD Identity Protection** op **maken**.
   
    ![Maken](./media/enable/04.png "Maken")
