# <a name="develop-secure-cloud-applications-on-azure"></a>Veilige Cloud toepassingen ontwikkelen op Azure
## <a name="abstract"></a>Samenvatting
Dit document is een algemene hand leiding voor de beveiligings vragen en-besturings elementen waarmee u rekening moet houden in elke fase van de levens cyclus van software ontwikkeling bij het ontwikkelen van toepassingen voor de Cloud. Als u deze concepten implementeert voordat u uw product uitgeeft, kunt u beter beveiligde software bouwen. De aanbevelingen die in dit artikel worden gepresenteerd, zijn afkomstig uit onze ervaring met Azure-beveiliging en de ervaringen van onze klanten.

Dit artikel is bedoeld als een resource voor software ontwerpers, ontwikkel aars en testers op alle niveaus die beveiligde Azure-oplossingen bouwen en implementeren.
