# <a name="service-bus-messaging-samples"></a>Voorbeelden van berichten over de bus
De Service Bus messaging monsters tonen de belangrijkste functies in [Service Bus messaging](https://azure.microsoft.com/services/service-bus/). Momenteel u de voorbeelden op de volgende plaatsen vinden:

| Programmeertaal | SDK of monsterlocatie |
| -------------------- | ----------------------- |
| .NET, Java en Beheer | https://github.com/Azure/azure-service-bus/ |
| Node.js | https://github.com/Azure/azure-sdk-for-js/tree/master/sdk/servicebus/service-bus/samples |
| Python | https://github.com/Azure/azure-sdk-for-python/tree/master/sdk/servicebus/azure-servicebus |

## <a name="service-bus-explorer"></a>Service Bus Explorer

Bovendien is de [Service Bus Explorer](https://github.com/paolosalvatori/ServiceBusExplorer) een voorbeeld dat wordt gehost op GitHub waarmee u verbinding maken met de naamruimte van een Service Bus-service en berichtenentiteiten eenvoudig beheren. De tool biedt geavanceerde functies, zoals import/ export functionaliteit, en de mogelijkheid om messaging entiteiten en relay diensten te testen. U vindt de volledige Bron en documentatie van Service Bus Explorer op [GitHub.](https://github.com/paolosalvatori/ServiceBusExplorer)
