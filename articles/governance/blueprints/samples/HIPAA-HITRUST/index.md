# <a name="overview-of-the-hipaa-hitrust-blueprint-sample"></a>Overzicht van het HIPAA HITRUST-blauwdrukvoorbeeld

De HIPAA HITRUST-blauwdrukvoorbeelden bieden een set beheerrichtlijnen op basis van [Azure Policy](../../../policy/overview.md) die bijdragen aan de [HIPAA HITRUST](https://www.hhs.gov/hipaa/for-professionals/security/laws-regulations/index.html)-attestatie. 

## <a name="blueprint-sample"></a>Blauwdrukvoorbeeld

Met behulp van deze blauwdrukvoorbeelden kunnen klanten een kernset beleidsregels implementeren voor elke Azure-architectuur waarvoor accreditatie of naleving van het HIPAA HITRUST-framework is vereist. De sectie over de [toewijzing van beheeropties](./control-mapping.md) biedt meer informatie over de beleidsregels die in dit initiatief zijn opgenomen, en hoe dit beleid kan bijdragen aan verschillende beheeropties die zijn gedefinieerd in het HIPAA HITRUST-framework.
