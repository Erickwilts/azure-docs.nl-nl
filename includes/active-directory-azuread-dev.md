> [!IMPORTANT]
> [Het Microsoft Identity Platform (v2.0)](/azure/active-directory/develop/v2-overview) bouwt voort [op het ontwikkelaarsplatform (v1.0) van Azure Active Directory (Azure AD)](../articles/active-directory/azuread-dev/about-microsoft-identity-platform.md). Met het Microsoft Identity Platform kunnen ontwikkelaars toepassingen maken waarbij gebruikers zich met alle Microsoft-identiteiten kunnen aanmelden en waarmee tokens worden opgehaald voor het aanroepen van Microsoft-API's, zoals Microsoft Graph, of API's die door ontwikkelaars zijn gemaakt.
> Deze inhoud is voor het oudere Azure AD v 1.0-eindpunt. We raden aan dat u het v2.0-eindpunt gebruikt voor nieuwe projecten. Raadpleeg voor meer informatie [Waarom bijwerken naar Microsoft Identity Platform (v2.0)?](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) en [Beperkingen van het Microsoft Identity-platform](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations).
