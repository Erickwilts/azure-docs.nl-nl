>[!NOTE]
> Dit artikel is onlangs bijgewerkt waarbij Log Analytics is vervangen door de term Azure Monitor-logboeken. Logboekgegevens worden nog steeds opgeslagen in een Log Analytics-werkruimte, en worden nog steeds verzameld en geanalyseerd met dezelfde Log Analytics-service. De terminologie wordt bijgewerkt om de rol van [logboeken in Azure Monitor](../articles/azure-monitor/platform/data-platform-logs.md) beter te weerspiegelen. Zie [Wijzigingen in Azure Monitor-terminologie](../articles/azure-monitor/terminology.md) voor meer informatie.
