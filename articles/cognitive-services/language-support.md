# <a name="natural-language-support-for-azure-cognitive-services"></a>Ondersteuning voor natuurlijke taal voor Azure Cognitive Services

Met Azure Cognitive Services kunt u toepassingen ontwikkelen die uw gebruikers zien, horen, spreken en begrijpen. Tussen deze services worden er meer dan drie dozijn talen ondersteund, zodat gebruikers op natuurlijke wijze met uw toepassing kunnen communiceren.

Dit artikel is onderverdeeld in twee secties. De eerste bevat een lijst met kern talen die doorgaans worden ondersteund in azure Cognitive Services. Het tweede gedeelte richt zich op extra taal ondersteuning voor elk van de services. Beschik baarheid van talen kan variëren per land/regio en op de markt.

## <a name="core-languages"></a>Kern talen

Deze kern talen worden ondersteund in azure Cognitive Services:

* Chinees
* Nederlands
* Frans
* Duits
* Italiaans
* Japans
* Koreaans ¹
* Portugees
* Spaans

> [!NOTE]
> ¹ LUIS, Video Indexer, Text Analytics en spraak naar tekst worden niet ondersteund.

## <a name="additional-language-availability-by-service"></a>Extra taal beschik baarheid per service

Deze tabellen markeren taal beschik baarheid per service categorie; uitgesloten kern talen. Gebruik de onderstaande koppelingen voor meer informatie over aanvullende taal ondersteuning en de beschik baarheid van landen/regio's en markten.
