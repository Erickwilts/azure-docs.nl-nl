# <a name="azure-policy-built-in-definitions-for-azure-signalr"></a>Ingebouwde Azure Policy-definities voor Azure SignalR

Deze pagina is een index van ingebouwde [Azure Policy](../governance/policy/overview.md)-beleidsdefinities voor Azure SignalR. Zie [Ingebouwde Azure Policy-definities](../governance/policy/samples/built-in-policies.md) voor aanvullende ingebouwde modules voor Azure Policy voor andere services.
