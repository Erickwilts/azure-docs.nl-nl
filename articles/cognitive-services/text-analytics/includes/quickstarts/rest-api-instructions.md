1. Kopieer de opdracht in een teksteditor.
2. Breng waar nodig de volgende wijzigingen in de opdracht aan:
    1. Vervang de waarde van `<your-text-analytics-key-here>` door uw sleutel.
    2. Vervang het eerste deel van de aanvraag-URL (`<your-text-analytics-endpoint-here>`) door uw eigen eindpunt-URL.
3. Open een opdrachtpromptvenster.
4. Plak de opdracht van de teksteditor in het opdrachtpromptvenster en voer de opdracht uit.
