# <a name="threat-modeling-tool-update-release-71607021---722019"></a>Threat Modeling Tool update release 7.1.60702.1 - 7/2/2019

Versie 7.1.60702.1 van de Microsoft Threat Modeling Tool (TMT) is uitgebracht op 2 juli 2019 en bevat de volgende wijzigingen:

- Verbeterde toegankelijkheid
- Opgeloste fouten
- Opname van een open-source community die stencilset heeft

## <a name="feature-changes"></a>Functiewijzigingen

### <a name="a-new-medical-devices-stencil-set-provided-by-the-open-source-community-is-available"></a>Een nieuwe medische apparaten stencil set die door de open-source gemeenschap is beschikbaar

Een stencilset voor het modelleren van medische hulpmiddelen is bijgedragen door de open-source community. Na het bijwerken wordt de nieuwe stencilset weergegeven in het vervolgkeuzemenu voor sjabloonselectie. Voor informatie over het bijdragen van stencils of inhoud aan sjablonen, controleert u de informatie op de [GitHub-pagina](https://github.com/Microsoft/threat-modeling-templates)van het project.

![Modelvalidatie, optie](./media/threat-modeling-tool-releases-71607021/tmt-template-selection.png)
