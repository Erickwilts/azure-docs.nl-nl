# <a name="scenario-apache-tez-application-hangs-in-azure-hdinsight"></a>Scenario: Apache Tez-toepassing hangt in Azure HDInsight

In dit artikel worden stappen voor het oplossen van problemen en mogelijke oplossingen voor problemen beschreven bij interactie met Azure HDInsight-clusters.

## <a name="issue"></a>Probleem

Na het indienen van de Apache Hive-taak, vanuit Tez-weergave, is de taakstatus 'Uitvoeren' maar lijkt geen vooruitgang te worden geboekt

## <a name="cause"></a>Oorzaak

Te veel ingediende vacatures; lange garenwachtrij.

## <a name="resolution"></a>Oplossing

Schaal het cluster, of gewoon wachten tot de garen wachtrij is uitgelekt.

Bepaalt `yarn.scheduler.capacity.maximum-applications` standaard het maximum aantal toepassingen dat wordt uitgevoerd of `10000`in behandeling is, en het standaard .
