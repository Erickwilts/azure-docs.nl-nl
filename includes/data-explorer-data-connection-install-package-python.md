## <a name="install-python-package"></a>Python-pakket installeren

Als u het Python-pakket voor Azure Data Explorer (Kusto) wilt installeren, opent u een opdrachtprompt met Python op zijn pad. Voer de volgende opdracht uit:

```python
pip install azure-common
pip install azure-mgmt-kusto
```
