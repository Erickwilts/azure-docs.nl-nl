# <a name="documentation-resources-for-cognitive-search-workloads"></a>Documentatie bronnen voor cognitieve Zoek werkbelastingen

Cognitieve Zoek opdrachten, die nu algemeen beschikbaar zijn, is een nieuwe verrijkings laag in Azure Search indexeren waarmee latente informatie in niet-tekst bronnen en niet-gedifferentieerde tekst wordt getransformeerd in volledige tekst Doorzoek bare inhoud in Azure Search.
