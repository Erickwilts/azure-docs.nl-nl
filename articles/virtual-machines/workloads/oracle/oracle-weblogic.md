# <a name="oracle-weblogic-server-azure-applications"></a>Oracle WebLogic Server Azure-toepassingen

## <a name="oracle-weblogic-server-is-a-scalable-enterprise-ready-java-ee-application-server"></a>Oracle WebLogic Server is een schaal bare, voor het bedrijf geschikte Java EE-toepassings server.

Oracle WebLogic Server is de eerste Cloud systeem eigen, zakelijke Java-platform toepassings server voor het ontwikkelen en implementeren van gedistribueerde zakelijke toepassingen met meerdere lagen. Azure WebLogic Server biedt u de mogelijkheid om Cloud computing te gebruiken door meer keuze en flexibiliteit te bieden voor WebLogic-migratie, waaronder lift-en Shift-toepassingen voor Azure-Clouds, met de kleinste inspanning en de grootste impact. De aanbiedingen bieden u de mogelijkheid om snel uw zakelijke toepassingen te starten door automatisch virtuele netwerk-, opslag-en Linux-resources in te richten, WebLogic Server te installeren, beveiliging in te stellen met een netwerk beveiligings groep, taak verdeling met Azure-app gateway, verificatie met Azure Active Directory en database connectiviteit te versnellen.

Er zijn vier aanbiedingen die kunnen voldoen aan verschillende scenario's: Eén knoop punt zonder een beheer server, één knoop punt met een beheer server, cluster en dynamisch cluster.  Het is raadzaam om hen een try-out te geven. de aanbiedingen zijn gratis beschikbaar.

_Deze aanbiedingen zijn uw eigen licentie_. Er wordt van uitgegaan dat u de juiste licenties hebt aangeschaft met Oracle en de juiste licentie hebt voor het uitvoeren van aanbiedingen in Microsoft Azure.

_Als u nauw keurig wilt werken met uw migratie scenario's met het technische team dat deze aanbiedingen ontwikkelt, kunt u gewoon op de knop [contact opnemen](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/oracle.oraclelinux-wls-cluster?tab=Overview) _ in de [Azure Marketplace](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/oracle.oraclelinux-wls-cluster?tab=Overview)drukken. Programma beheerders, Architects en technici zullen binnenkort een back-out maken en samen werking initiëren.

### <a name="oracle-weblogic-server-single-node"></a>Oracle WebLogic-Server met één knoop punt

Deze aanbieding biedt een enkele virtuele machine en installeert Oracle WebLogic Server. Er wordt geen domein gemaakt of de beheer server wordt gestart. Dit is handig voor scenario's met een zeer aangepaste domein configuratie.

### <a name="oracle-weblogic-server-with-admin-server"></a>Oracle WebLogic Server met de-beheer server

Deze aanbieding biedt een enkele virtuele machine en installeert Oracle WebLogic Server. Er wordt een domein gemaakt en de beheer server wordt gestart, waarmee u het domein kunt beheren.

### <a name="oracle-weblogic-server-cluster"></a>Oracle WebLogic Server-cluster

Met deze aanbieding maakt u een Maxi maal beschikbaar cluster met virtuele machines van Oracle WebLogic Server. De-beheer server en alle beheerde servers worden standaard gestart, zodat u het domein kunt beheren.

### <a name="oracle-weblogic-server-dynamic-cluster"></a>Dynamische Oracle WebLogic Server-cluster

Met deze aanbieding maakt u een Maxi maal beschikbaar en schaalbaar dynamisch cluster met virtuele machines van Oracle WebLogic Server. De-beheer server en alle beheerde servers worden standaard gestart, zodat u het domein kunt beheren.
