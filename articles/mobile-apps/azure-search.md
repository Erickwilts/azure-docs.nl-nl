# <a name="ai-powered-cloud-service-with-azure-search"></a>Een AI-Cloud service met Azure Search
[Azure Search](https://azure.microsoft.com/services/search/) is een Cloud oplossing op basis van een zoek opdracht als service die ontwikkel aars api's en hulpprogram ma's biedt voor het toevoegen van een uitgebreide zoek ervaring via persoonlijke, heterogene inhoud in web-, mobiele en zakelijke toepassingen. Met uw aangepaste code wordt gegevens opname (indexering) aangeroepen om een index te maken en te laden. Aan de andere kant worden met de toepassings code query aanvragen aangevraagd en worden antwoorden verwerkt. De zoek ervaring wordt gedefinieerd in uw client met behulp van de functionaliteit van Azure Search, waarbij de query wordt uitgevoerd via een persistente index die u maakt, eigenaar en opslaat in uw service.

## <a name="azure-search-features"></a>Azure Search functies
- **Zoek opdracht in tekst zonder** opmaak met volledige tekst zoeken, eenvoudige query syntaxis en de Lucene-query syntaxis.
- **Eenvoudige Score profielen** om relevantie te model leren als een functie van waarden in de documenten zelf.
- Met **Geo-search** kunnen gebruikers gegevens verkennen op basis van de nabijheid van een Zoek resultaat naar een fysieke locatie.
- **Gebruikers ervaring heeft functies** met automatisch aanvullen, zoek suggesties, sorteren en paginering.
- **Cognitieve Zoek opdrachten** voor afbeeldings-en tekst analyse kunnen worden toegepast op een indexerings pijplijn om tekst gegevens uit onbewerkte inhoud te halen.
- **Kennis archief** om verrijkingen op te slaan die tijdens het indexeren zijn gemaakt.
- Maak verbinding met Azure SQL Database, Azure Cosmos DB of Azure Blob Storage als **gegevens bronnen** voor het extra heren van Doorzoek bare inhoud in primaire gegevens archieven.
- Gebruik de **wizard gegevens importeren** om Indexeer functies, **index Designer** te configureren om een index op te stellen en **Zoek Verkenner** om query's te testen en Score profielen te verfijnen.

Hier vindt u een gedetailleerde beschrijving van Azure Search- [functies](/azure/search/search-what-is-azure-search#feature-descriptions).
