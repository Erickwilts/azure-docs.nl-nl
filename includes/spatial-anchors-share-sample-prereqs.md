## <a name="prerequisites"></a>Vereisten

Het volgende moet zijn geïnstalleerd om deze zelfstudie te voltooien:

* Lees het [overzicht van Azure Spatial Anchors](../articles/spatial-anchors/overview.md).
* Voltooid een van de [5 minuten Quickstarts](../articles/spatial-anchors/index.yml).
* Basiskennis over C# en Unity.
* Basiskennis over <a href="https://developers.google.com/ar/discover/" target="_blank">ARCore</a> als je Android of <a href="https://developer.apple.com/arkit/" target="_blank">ARKit</a> wilt gebruiken als je iOS wilt gebruiken.
* Een Windows-computer waarop <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017</a> of hoger is geïnstalleerd met de **ASP.NET en webdevelopment-workload.**
* De [.NET Core 2.2 SDK](https://dotnet.microsoft.com/download).
* Een of meer apparaten (iOS of Android) waarop een app kan worden geïmplementeerd en uitgevoerd.
  * Als je Android gebruikt, moet je het:
    * <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3</a> of hoger, <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2019.1 of 2019.2</a>en <a href="https://git-scm.com/download/win" target="_blank">Git voor Windows</a> geïnstalleerd op uw Windows-computer.
    * Een <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">Android-apparaat dat geschikt is voor ontwikkelaars</a> en <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">ARCore.</a>
  * Als u iOS gebruikt, moet u het:
    * Een macOS-computer waarop <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10</a> of hoger, <a href="https://cocoapods.org" target="_blank">CocoaPods</a>en <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2019.1 of 2019.2</a> zijn geïnstalleerd.
    * Een <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">ARKit-compatibel iOS-apparaat</a> met ontwikkelaar.
    * Git geïnstalleerd via Homebrew. Voer de volgende opdracht in op `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"` één `brew install git`regel in de terminal: Voer deze uit.
