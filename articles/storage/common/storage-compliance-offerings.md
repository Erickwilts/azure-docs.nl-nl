# <a name="azure-storage-compliance-offerings"></a>Nalevingsaanbiedingen voor Azure Storage

Microsoft Azure & Azure Storage bieden de meest uitgebreide set certificeringen en verklaringen van elke cloudserviceprovider om organisaties te helpen voldoen aan nationale, regionale en branchespecifieke vereisten voor het verzamelen en gebruiken van gegevens van personen.

U vindt hieronder nalevingsaanbiedingen op Azure Storage om ervoor te zorgen dat uw service is gereguleerd in het gebruik van Azure Storage-service. Ze zijn van toepassing op de volgende Azure Storage-aanbiedingen: Blobs, Files, Queues, Tables, Disks, Cool Storage en Premium Storage.
