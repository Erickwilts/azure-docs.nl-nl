# <a name="azure-powershell-samples-for-azure-data-factory"></a>Azure PowerShell-voorbeelden voor Azure Data Factory

| |  |
|---|---|
|**Gegevens kopiëren**||
|[Blobs kopiëren van een map naar een andere map in een Azure Blob Storage](scripts/copy-azure-blob-powershell.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Met dit PowerShell-script worden blobs kopieën van een map in Azure Blob Storage naar een andere map in dezelfde Blob-opslag. |
