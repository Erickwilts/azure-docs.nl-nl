[Met Azure App Service](/azure/app-service/overview) u code bouwen en hosten. Deze code kan worden in de vorm van web-apps, mobiele backends, en RESTful API's. Met Behulp van App Service u uw code ontwikkelen met behulp van de programmeertaal van uw keuze zonder de infrastructuur te beheren. App Service ondersteunt zowel Windows als Linux. Geautomatiseerde implementaties van elke Git repo worden ondersteund, waaronder GitHub en Azure DevOps.
