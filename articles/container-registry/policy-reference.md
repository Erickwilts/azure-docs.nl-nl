# <a name="azure-policy-built-in-definitions-for-azure-container-registry"></a>Ingebouwde Azure Policy-definities voor Azure Container Registry

Deze pagina is een index van ingebouwde [Azure Policy](../governance/policy/overview.md)-beleidsdefinities voor Azure Container Registry. Zie [Ingebouwde Azure Policy-definities](../governance/policy/samples/built-in-policies.md) voor aanvullende ingebouwde modules voor Azure Policy voor andere services.
