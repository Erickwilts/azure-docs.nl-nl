## <a name="test"></a>De functie in azure controleren

U kunt een webbrowser gebruiken om de geïmplementeerde functie te controleren.  Kopieer de URL, met inbegrip van de functie sleutel, naar de adres balk van uw webbrowser. Voeg de query reeks `&name=<yourname>` toe aan de URL voordat u de aanvraag uitvoert.

![Een webbrowser gebruiken om de functie aan te roepen.](./media/functions-test-function-code/functions-azure-cli-function-test-browser.png)  

U kunt ook krul gebruiken om de geïmplementeerde functie te controleren. Voeg met behulp van de URL, met inbegrip van de functie sleutel, die u hebt gekopieerd uit de vorige stap, de query reeks toe `&name=<yourname>` toe aan de URL.

![Krul gebruiken om de functie in azure aan te roepen.](./media/functions-test-function-code/functions-azure-cli-function-test-curl.png)
