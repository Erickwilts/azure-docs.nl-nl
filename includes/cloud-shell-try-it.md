## <a name="use-azure-cloud-shell"></a>Azure Cloud Shell gebruiken

Azure host Azure Cloud Shell, een interactieve shell-omgeving die u via uw browser gebruiken. U Bash of PowerShell met Cloud Shell gebruiken om met Azure-services te werken. U de vooraf geïnstalleerde opdrachten van Cloud Shell gebruiken om de code in dit artikel uit te voeren zonder dat u iets hoeft te installeren op uw lokale omgeving.

Ga als een start van Azure Cloud Shell:

| Optie | Voorbeeld/koppeling |
|-----------------------------------------------|---|
| Selecteer **Nu proberen** in de rechterbovenhoek van een codeblok. Als u **Try It selecteert,** wordt de code niet automatisch naar Cloud Shell gekopieerd. | ![Voorbeeld van Try It voor Azure Cloud Shell](./media/cloud-shell-try-it/hdi-azure-cli-try-it.png) |
| Ga [https://shell.azure.com](https://shell.azure.com)naar of selecteer de knop **Cloud Shell starten** om Cloud Shell in uw browser te openen. | [![Cloud Shell in een nieuw venster lanceren](media/cloud-shell-try-it/hdi-launch-cloud-shell.png)](https://shell.azure.com) |
| Selecteer de knop **Cloud Shell** op de menubalk rechtsboven in de [Azure-portal.](https://portal.azure.com) | ![Knop Cloud Shell in de Azure Portal](./media/cloud-shell-try-it/hdi-cloud-shell-menu.png) |

Ga als volgt te werk om de code in dit artikel uit te voeren in Azure Cloud Shell:

1. Start Cloud Shell.

1. Selecteer de knop **Kopiëren** in een codeblok om de code te kopiëren.

1. Plak de code in de Cloud Shell-sessie door **Ctrl**+**Shift**+**V** op Windows en Linux te selecteren of door **Cmd**+**Shift**+**V** op macOS te selecteren.

1. Selecteer **Enter** om de code uit te voeren.
