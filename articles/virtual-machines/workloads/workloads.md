# <a name="workloads-on-azure-virtual-machines"></a>Werk belastingen op Azure Virtual Machines

Virtuele machines in azure kunnen veel werk belastingen ondersteunen. Elke workload kan gespecialiseerde informatie hebben over het optimale ontwerp voor implementaties op Vm's in Azure. Hier volgen enkele richt lijnen die we kunnen aanbieden voor gespecialiseerde workloads.

- [High Performance Computing](./hpc/overview.md) in azure maakt gebruik van virtuele machines die zijn geoptimaliseerd voor CPU of GPU om complexe, computerintensieve problemen op te lossen.
- Met [SAP](./sap/get-started.md) in azure kunnen uw essentiële SAP-werk belastingen en-scenario's worden uitgevoerd op een schaalbaar, compatibel en op ondernemings niveau bewezen platform.
- [Oracle](./oracle/oracle-considerations.md) biedt ondersteuning voor het uitvoeren van Oracle DB 12,1 Standard-en Enter prise-edities in azure op installatie kopieën van virtuele machines op basis van Oracle Linux. 
- [Mainframe opnieuw hosten](./mainframe-rehosting/overview.md) 



**Volgende stappen**

Krijg kennis met Azure Vm's met de Snelstartgids voor Azure VM voor [Linux](../linux/quick-create-cli.md) en [Windows](../windows/quick-create-powershell.md).
