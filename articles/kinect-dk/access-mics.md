# <a name="access-azure-kinect-dk-microphone-input-data"></a>Invoer gegevens van Azure Kinect DK-microfoon openen

De Quick starts van de [Speech SDK](https://docs.microsoft.com/azure/cognitive-services/speech-service) bieden voor beelden van het gebruik van de Azure Kinect DK-microfoon matrix in verschillende programmeer talen.
Zie bijvoorbeeld het herkennen van **spraak in C++ in Windows met behulp van de Quick Start van de Speech SDK** . De code is beschikbaar [via github](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/cpp).

Open ook de microfoon matrix via de Windows-API. Raadpleeg de volgende documenten voor meer informatie over de Windows-documentatie:

* [Windows Audio-architectuur](https://docs.microsoft.com/windows-hardware/drivers/audio/windows-audio-architecture)
* [Documentatie over Windows. media. Capture](https://docs.microsoft.com/uwp/api/Windows.Media.Capture)
* [Zelf studie voor het vastleggen van webcams](https://docs.microsoft.com/windows/uwp/audio-video-camera/basic-photo-video-and-audio-capture-with-mediacapture)
* [USB-audio gegevens](https://docs.microsoft.com/windows-hardware/drivers/audio/usb-2-0-audio-drivers)

U kunt ook de hardware-specificatie van de [microfoon matrix](hardware-specification.md#microphone-array)bekijken.
