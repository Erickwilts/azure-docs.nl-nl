## <a name="set-properties"></a>Eigenschappen instellen

U ervoor kiezen om bepaalde eigenschappen toe te voegen bij het opslaan van uw ruimtelijke ankers in de cloud. Zoals het type object dat wordt opgeslagen of basiseigenschappen zoals of het moet worden ingeschakeld voor interactie. Dit kan handig zijn bij ontdekking: u het object onmiddellijk weergeven voor de gebruiker, bijvoorbeeld een fotolijst met lege inhoud. Vervolgens krijgt een andere download op de achtergrond extra statusdetails, bijvoorbeeld de afbeelding die in het frame moet worden weergegeven.
