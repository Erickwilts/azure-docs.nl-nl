# <a name="third-party-software-compatible-with-azure-ad-domain-services"></a>Software van derden die compatibel is met Azure AD Domain Services

## <a name="one-identity-authentication-services"></a>Eén identiteits verificatie services

![Eén identiteit](./media/compatible-software-icons/OneIdentity.png)

Met één identiteits verificatie service kunnen UNIX-, Linux-en Mac OS X-systemen gebruikmaken van de toegang, verificatie en autorisatie van de bestaande Active Directory-infra structuur (AD) van een organisatie. Verificatie services ondersteunt nu Azure Active Directory Domain Services het inschakelen van niet-Windows-bronnen om gebruik te maken van hetzelfde platform van de volgende generatie die al door uw bestaande SaaS-oplossingen wordt gebruikt.

[Eén identiteits verificatie services](https://www.quest.com/products/authentication-services/)

[KB-artikel: gebruik één identiteits verificatie service met Azure AD Domain Services](https://support.quest.com/authentication-services/kb/208427)


## <a name="red-hat-enterprise-linux"></a>Red Hat Enterprise Linux

![Red Hat](./media/compatible-software-icons/RedHat.png)

Red Hat Enterprise Linux is het toonaangevende Enter prise Linux-platform dat is gebouwd om te voldoen aan de behoeften van de moderne onderneming van vandaag. Red Hat Enterprise Linux is de voorkeurs keuze voor werk belastingen van de Enter prise Linux Virtual Machine (VM) op Microsoft Azure. Red Hat Enterprise Linux is een open, betrouwbaar en veilig platform dat is ontworpen voor klanten die implementatie flexibiliteit voor hun bedrijfskritische werk belastingen willen: van het Data Center tot de Azure-Cloud die wordt ondersteund door nauw geïntegreerde, hoogwaardige ondersteuning op bedrijfs niveau Red Hat en micro soft.

[Red Hat Enterprise Linux 7,2 VM toevoegen aan Azure AD Domain Services](active-directory-ds-admin-guide-join-rhel-linux-vm.md)
