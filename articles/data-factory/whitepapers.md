# <a name="azure-data-factory-whitepapers"></a>Whitepapers voor Azure Data Factory 

[!INCLUDE[appliesto-adf-xxx-md](includes/appliesto-adf-xxx-md.md)]

Met whitepapers u Azure Data Factory op een dieper niveau verkennen. In dit artikel vindt u een lijst met beschikbare whitepapers voor Azure Data Factory.

| **Whitepaper** | **Beschrijving** |
| --- | --- |
|[Azure Data Factory—Gegevensintegratie in de cloud](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | In dit document wordt beschreven hoe Azure Data Factory u in staat kan stellen een modern datawarehouse te bouwen, geavanceerde analyses mogelijk te maken om intelligente SaaS-toepassingen te stimuleren en uw SQL Server Integrations Services-pakketten naar Azure te tillen.|
|[Gegevensmigratie van on-premises relationele gegevensmagazijn naar Azure met Azure Data Factory](https://azure.microsoft.com/resources/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/) | Dit document richt zich op de complexiteit van het migreren van tientallen TB-gegevens uit bestaande on-premises relationele gegevensmagazijn (bijvoorbeeld Netezza, Oracle, Teradata, SQL-server) naar Azure (bijvoorbeeld Blob Storage of Azure Data Lake Storage) met Azure Data Factory. De uitdagingen en best practices worden geïllustreerd rond veerkracht, prestaties, schaalbaarheid, beheer en beveiliging voor het reis naar Azure door Azure Data Factory. |
|[SAP-gegevensintegratie met Azure Data Factory](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | In dit document wordt een overzicht gegeven van de huidige ondersteuning van Azure Data Factory voor SAP-gegevensintegratie, inclusief doelscenario's, SAP-connectoropties en vergelijking voor verschillende vereisten, en introductie op elke SAP-connector in Data Factory. | 
|[Azure Data Factory: SSIS in de cloud](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| In dit document wordt verteld waarom u uw bestaande SSIS-workloads wilt migreren naar Azure Data Factory en algemene overwegingen en problemen wilt aanpakken. We nemen u vervolgens de technische details van het maken van een Azure-SSIS IR door en laten u vervolgens zien hoe u uw pakketten uploadt, uitvoert en bewaakt via Azure Data Factory met behulp van de tools die u waarschijnlijk kent, zoals SQL Server Management Studio (SSMS). |
|[Azure-gegevensfabriek –Parameters doorgeven](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| In dit artikel u zien hoe u parameters tussen een pijplijn en activiteit en tussen de activiteiten doorgeven. |
|[Azure-gegevensfabriek –DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| Deze whitepaper bevat enkele van de aanbevolen procedures rond continue integratie en implementatie van Azure Data Factory. |
