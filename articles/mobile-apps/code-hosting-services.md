# <a name="cloud-hosted-source-code-management-services"></a>In de Cloud gehoste bron code beheer Services
Als u ontwikkel teams hebt met meerdere team leden die aan dezelfde code basis werken, moet u een geschikte plaats vinden om uw code te hosten. Door de gegevens op te slaan in de Cloud en een gecentraliseerde opslag plaats te gebruiken, kan iedereen de code bestanden uploaden, bewerken en beheren, communiceren met andere ontwikkel aars op projecten, en kan de code eenvoudig toegankelijk zijn, waar u ook bent, op voor waarde dat u een Internet verbinding hebben.

Het hosten van Clouds is veel eenvoudiger dan on-premises opties, omdat het minder hardwareconfiguratie vereist en organisaties in staat stelt het implementatie proces op een flexibeler manier uit te voeren.

## <a name="benefits-of-hosting-source-code-in-the-cloud"></a>Voor delen van het hosten van bron code in de Cloud
- **Gecentraliseerde Cloud opslag** om uw gegevens vanaf elke locatie weer te geven en te beheren.
- Zorgt voor **betere samen werking en verruiming van code** in teams om code bij te houden en projecten te beheren om de continue levering van uitstekende software te garanderen.
- **Doe** mee aan een **bijdrage leveren** aan uw projecten.
- **Snellere release cyclus** als uw team sneller en eenvoudig kan **bijdragen** aan uw projecten.
- **Verminder de kosten** zonder dat u zich zorgen hoeft te maken over het onderhouden van uw eigen hardware, servers, VPN, enzovoort.

Gebruik de volgende services voor het hosten van uw toepassings gegevens in de Cloud.

## <a name="github"></a>GitHub
[Github](https://github.com/) is een open-source service voor het hosten van opslag plaatsen die als host fungeert voor broncode projecten in verschillende programmeer talen en waarmee de verschillende wijzigingen die in elke iteratie worden aangebracht, worden bijgehouden.

**Belangrijkste functies**
- **Code hosting** met al uw code op één plek. Persoonlijke, open bare of open source, alle opslag plaatsen zijn voorzien van hulpprogram ma's die u helpen bij het hosten, versie en vrijgeven van code.
- Dankzij de naadloze beoordeling van **code** en de ingebouwde beoordelings hulpmiddelen wordt de code een essentieel onderdeel van elk teams proces gecontroleerd.
    - Filialen beveiligen, wijzigingen Voorst Ellen en beoordelingen opvragen.
    - Bekijk het verschil en de opmerking in context en ontvang duidelijke feedback.
- U kunt in een vroeg stadium een coördinaat maken en meer doen met de **project management tools**van github.
    - Bekijk de grote afbeelding van het project.
    - **Taak borden** die zich rechts naast uw code bevinden in github.
    - **Kaarten slepen en neerzetten** zodat u problemen kunt toewijzen of aanvragen kunt ophalen aan team leden.
    - **Mijl palen** voor het organiseren en bijhouden van de voortgang.
    - **Notities** voor het vastleggen van ideeën die nuttig kunnen zijn, maar die geen deel uitmaken van een bepaald probleem of een pull-aanvraag.
- U kunt eenvoudig de juiste hulpprogram ma's ontdekken en kiezen voor betere communicatie en automatisering van werk door toepassingen te kopen van **[github Marketplace](https://github.com/marketplace)** .
- **Team beheer** om teams eenvoudig te beheren en te verg Roten, **gebruikers rollen** om teams en toegangs machtigingen te organiseren, hulpprogram ma's voor **discussie-threads** om discussies over onderwerp en team gericht te houden en **richt lijnen** van de Community snel in te stellen nieuwe team leden met een account.
- **Blader door** **populaire projecten** om ze te volgen.
- **Netwerk en leer** van anderen in de branche.

## <a name="azure-devops"></a>Azure DevOps
[Azure DevOps](https://azure.microsoft.com/services/devops/) ondersteunt [Azure opslag plaatsen](https://azure.microsoft.com/services/devops/repos/) en [Team Foundation Version Control (TFVC)](https://docs.microsoft.com/azure/devops/repos/tfvc/index?view=azure-devops) als broncode beheer opties. Het heeft onbeperkte gratis privé-opslag plaatsen met revisies op het samen stellen van code, Geavanceerd bestands beheer, zoeken naar code en vertakkings beleid om code van hoge kwaliteit te garanderen. Azure opslag plaatsen is geschikt voor kleine projecten en grote organisaties die systeem eigen AAD-ondersteuning en geavanceerde beleids regels nodig hebben.
    
**Belangrijkste functies**
- **Onbeperkte in de Cloud gehoste Git-bron code opslagplaats** voor uw open bare en persoonlijke opslag plaatsen
    - Ondersteuning voor een Git-client.
    - Webhooks en API-integratie.
- **Uw volgende Build** uit een opslag plaatsen-pull-aanvraag starten
    - Werk samen om betere code te bouwen met behulp van threaded discussies en continue integratie voor elke wijziging.
    - Zorg voor een continue integratie/continue levering (CI/CD) zodat u Azure Pipelines of uw eigen hulpprogramma's kunt gebruiken om automatisch builds, tests en implementaties te activeren voor elke voltooide pull-aanvraag.
    - Bescherm uw code kwaliteit met vertakkings beleid.
- **Gecentraliseerd versie beheer met versie beheer van Team Foundation**, inclusief code controle.
- **Maak verbinding met uw code** met behulp van Xcode, eclips, IntelliJ, Android Studio, Visual Studio, Visual Studio code en meer.
- **Krachtige semantische code zoeken**.
- Het **bedrijf is klaar** voor de systeem eigen integratie met Azure Active Directory (AD), waardoor het beheer van de toegang tot uw code opslagplaatsen wordt vereenvoudigd.
- **Zorg voor code kwaliteit** met branch-beleids regels, zoals het minimale aantal code beoordelingen, het vereisen van geslaagde builds en het afdwingen van Git-samenvoeg strategieën.
- **Verbind uw favoriete ontwikkel omgeving** om toegang te krijgen tot opslag plaatsen en uw werk te beheren.
