# <a name="azure-application-offer-settings-tab"></a>Tabblad Instellingen voor Azure-toepassingsaanbieding

In dit artikel wordt beschreven hoe u de aanbiedingsinstellingen voor een Azure-toepassing configureert.

De pagina **Azure Applications > Nieuwe aanbieding** wordt geopend met de focus op het tabblad **Aanbiedingsinstellingen.** Een sterretje (*) dat aan de veldnaam is toegevoegd, geeft aan dat dit vereist is.

![Identiteitsformulier aanbieden](./media/azureapp-offer-settings-tab.png)

## <a name="offer-identity-settings"></a>Identiteitsinstellingen aanbieden

Onder **Aanbiedingsidentiteit**moet u informatie verstrekken voor de velden die in de volgende tabel worden beschreven.  

|    Veld         |       Beschrijving                                                            |
|  ---------       |     ---------------                                                          |
| **Aanbiedings-id\***       | Een unieke id (binnen een uitgeversprofiel) voor de aanbieding. Deze id is zichtbaar in product-URL's en insights-rapporten. Het heeft een maximale lengte van 50 tekens en kan alfanumerieke letters en streepjes (-) gebruiken. (De id kan niet eindigen met een streepje.) **Let op:** Dit veld kan niet worden gewijzigd nadat een aanbieding live is gegaan. <br> Als Contoso bijvoorbeeld een aanbieding publiceert met **voorbeeldcontainer**voor aanbiedings-id, krijgt u de URL `https://azuremarketplace.microsoft.com/marketplace/apps/contoso.sample-container?tab=Overview`van Azure Marketplace toegewezen . |
| **Publisher-id\***     | De unieke id van uw organisatie in de Azure Marketplace. Al uw aanbiedingen moeten worden gekoppeld aan uw uitgever-id. Deze waarde kan niet worden gewijzigd nadat de aanbieding is opgeslagen. |
| **Naam\***          | De weergavenaam voor uw aanbieding. Deze naam wordt weergegeven in de Azure Marketplace en in de Cloud Partner Portal. De naam mag maximaal 50 tekens bevatten. Wij raden u aan een herkenbare merknaam voor uw product te gebruiken. Vermeld de naam van uw organisatie niet, tenzij uw product op die manier op de markt wordt gebracht. Als u deze aanbieding op de markt zet in andere websites en publicaties, zorg er dan voor dat de naam in alle publicaties precies hetzelfde is. |
|  |  |

Selecteer **Opslaan** om uw aanbiedingsinstellingen op te slaan.

## <a name="next-steps"></a>Volgende stappen
