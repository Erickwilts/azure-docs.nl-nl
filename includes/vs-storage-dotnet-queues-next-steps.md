Nu u de basisbeginselen van Azure-wachtrijopslag hebt geleerd, volgt u deze koppelingen om meer te weten te komen over complexere opslagtaken.

* Bekijk de naslagdocumentatie van de wachtrijservice in de [Azure Storage Client Library voor .NET-referentie](https://go.microsoft.com/fwlink/?LinkID=390731) voor volledige details over beschikbare API's.
* Meer informatie over het gebruik van wachtrijopslag bij [Aan de slag met Azure Queue-opslag met .NET](../articles/storage/queues/storage-dotnet-how-to-use-queues.md)
* Ga naar [Wat is de Azure WebJobs SDK](https://github.com/Azure/azure-webjobs-sdk/wiki) voor meer informatie over het vereenvoudigen van de code die u schrijft om met Azure Storage te werken
* Bekijk meer functiehandleidingen voor informatie over aanvullende mogelijkheden voor het opslaan van gegevens in Azure.
  * Zie Aan de slag [met Azure Table-opslag met .NET](../articles/cosmos-db/table-storage-how-to-use-dotnet.md).
  * Als u wilt werken met Azure Storage Blobs, [gaat u aan de slag met Azure Blob-opslag met .NET](../articles/storage/blobs/storage-dotnet-how-to-use-blobs.md).
  * Zie [Verbinding maken met SQL Database met .NET (C#)](../articles/sql-database/sql-database-develop-dotnet-simple.md)als u relationele gegevens wilt opslaan.
