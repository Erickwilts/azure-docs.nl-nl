### <a name="run-multiple-containers-on-the-same-host"></a>Meerdere containers op dezelfde host worden uitgevoerd

Als u van plan bent om uit te voeren van meerdere containers met blootgestelde poorten, zorg ervoor dat elke container uitvoeren met een andere poort weergegeven. Bijvoorbeeld: de eerste container op poort 5000- en de tweede container uitvoeren op poort 5001.

U kunt deze container en een andere Azure Cognitive Services-container wordt uitgevoerd op de HOST bij elkaar hebben. U kunt ook meerdere containers van de dezelfde Cognitive Services-container die wordt uitgevoerd hebben.
