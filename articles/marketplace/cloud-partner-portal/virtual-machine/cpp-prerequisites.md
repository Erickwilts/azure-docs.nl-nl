# <a name="virtual-machine-prerequisites"></a>Vereisten voor virtuele machines

In dit artikel worden zowel de technische als zakelijke vereisten weergegeven waaraan u moet voldoen voordat u een VM-aanbieding naar de [Azure Marketplace](https://azuremarketplace.microsoft.com/marketplace/)publiceren.  Als u dit nog niet hebt gedaan, bekijkt u de [handleiding voor het publiceren van aanbiedingen voor virtuele machines](../../marketplace-virtual-machines.md).


## <a name="technical-requirements"></a>Technische vereisten

De technische vereisten voor het publiceren van een VM-oplossing (virtual machine) zijn eenvoudig:

- U moet een actief Azure-account hebben. Als u er geen hebt, u zich aanmelden op de [Microsoft Azure-site.](https://azure.microsoft.com)  
- U moet een omgeving hebben die is geconfigureerd om de ontwikkeling van Windows- of Linux VM te ondersteunen.  Zie de bijbehorende VM-documentatiesite voor meer informatie:
    - [Documentatie voor virtuele Linux-machines](https://docs.microsoft.com/azure/virtual-machines/linux/)
    - [Documentatie over virtuele Linux-machines](https://docs.microsoft.com/azure/virtual-machines/windows/)

- Uw bedrijf (of haar dochteronderneming) moet zich bevinden in een verkoop-uit-land/regio die wordt ondersteund door de Azure Marketplace.  Zie [Microsoft Azure Marketplace-deelnamebeleid](https://azure.microsoft.com/support/legal/marketplace/participation-policies/)voor een actuele lijst met deze landen/regio's .

- Je bent verantwoordelijk voor het op een commercieel redelijke manier beschikbaar stellen van technische ondersteuning aan klanten. Deze ondersteuning kan gratis, betaald of via communitybenaderingen zijn.
- U bent verantwoordelijk voor het verlenen van licenties voor uw software en eventuele afhankelijkheden van software van derden.
- U moet inhoud leveren die voldoet aan de criteria voor uw aanbod om te worden aangeboden op Azure Marketplace en in de Azure-portal. <!-- TD: Meaning/links? -->
