#### <a name="c-nuget-package"></a>C# NuGet-pakket

De .NET Speech SDK kan worden geïnstalleerd via de **.net core SLI** met de volgende `dotnet add` opdracht.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

De .NET Speech SDK kan worden geïnstalleerd via **Package Manager** met de volgende `Install-Package` opdracht.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
