## <a name="create-an-azure-resource"></a>Een Azure-resource maken

Begin met het gebruik van de Bing Image Search-API door een van de volgende Azure-resources te maken.

### <a name="bing-search-v7-resource"></a>[Bing Search v7-resource](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesBingSearch-v7)
   * Beschikbaar via de Azure-portal totdat u de resource verwijdert.
   * Gebruik de gratis prijscategorie om de service uit te proberen, en voer later een upgrade uit naar een betaalde categorie voor productie.

### <a name="multi-service-resource"></a>[Resource voor meerdere services](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAllInOne)
   * Beschikbaar via de Azure-portal totdat u de resource verwijdert.  
   * Gebruik dezelfde sleutel en hetzelfde eindpunt voor uw toepassingen, op meerdere Cognitive Services.
