# <a name="what-tools-are-included-on-the-azure-data-science-virtual-machine"></a>Welke hulpprogram ma's zijn opgenomen in de Azure Data Science Virtual Machine?

De Data Science Virtual Machine is de eenvoudigste manier om gegevens te verkennen en machine learning in de Cloud uit te voeren. De data Science Virtual Machines zijn vooraf geconfigureerd met het volledige besturings systeem, beveiligings patches, stuur Programma's en populaire software voor gegevens wetenschap en ontwikkeling. U kunt de hardware-omgeving kiezen, variërend van goedkope CPU-georiënteerde computers tot zeer krachtige computers met meerdere Gpu's, NVMe-opslag en grote hoeveel heden geheugen. Voor computers met Gpu's worden alle Stuur Programma's geïnstalleerd, alle machine learning frameworks zijn de versie die overeenkomt met de GPU-compatibiliteit en de versnelling is ingeschakeld in alle toepassings software die Gpu's ondersteunt.

Het Data Science Virtual Machine wordt geleverd met de meest nuttige hulpprogram ma's voor gegevens wetenschap die vooraf zijn geïnstalleerd:
