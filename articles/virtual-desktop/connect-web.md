# <a name="connect-with-the-web-client"></a>Verbinding maken met de webclient

Met de webclient hebt u toegang tot uw Windows Virtual Desktop-bronnen vanuit een webbrowser zonder het langdurige installatieproces.

>[!NOTE]
>De webclient heeft momenteel geen mobiele OS-ondersteuning.

## <a name="supported-operating-systems-and-browsers"></a>Ondersteunde besturingssystemen en browsers

## <a name="access-remote-resources-feed"></a>Toegang tot externe bronnenfeed

Navigeer in een browser naar de [Windows Virtual Desktop-webclient](https://rdweb.wvd.microsoft.com/webclient) en meld u aan met uw gebruikersaccount.

>[!NOTE]
>Als u zich al hebt aangemeld met een ander Azure Active Directory-account dan het account dat u wilt gebruiken voor Windows Virtual Desktop, moet u zich afmelden of een priv√©browservenster gebruiken.
