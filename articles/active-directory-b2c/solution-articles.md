# <a name="solutions-and-training-for-azure-active-directory-b2c"></a>Oplossingen en training voor Azure Active Directory B2C

Azure Active Directory B2C (Azure AD B2C) stelt organisaties in staat om bedrijfsoplossingen te implementeren waarmee ze verbinding kunnen maken met hun klanten. De volgende oplossingsgidsen en -training zijn downloadbare documenten die u door deze oplossingen leiden.

| Titel | Beschrijving |
| ----- | ----------- |
| [Klantidentiteitsbeheer met Azure AD B2C](https://channel9.msdn.com/Shows/On-NET/Customer-Identity-Management-with-Azure-AD-B2C) | In dit overzicht van de service[@jainparakh](https://twitter.com/jainparakh)geeft Parakh Jain ( ) van het Azure AD B2C-team ons een overzicht van hoe de service werkt, en laat ook zien hoe we B2C snel kunnen verbinden met een ASP.NET Core-toepassing. |
| [Voordelen van het gebruik van Azure AD B2C](https://aka.ms/b2coverview) | Inzicht in de voordelen en algemene scenario's van Azure AD B2C en hoe uw toepassing(en) gebruik kunnen maken van deze CIAM-service. |
| [Expertise verwerven in Azure AD B2C: een cursus voor ontwikkelaars](https://aka.ms/learnAADB2C) | Deze end-to-end cursus neemt ontwikkelaars mee door een complete reis op het ontwikkelen van toepassingen met Azure AD B2C als verificatiemechanisme. Tien diepgaande modules met labs bestrijken alles, van het instellen van een Azure-abonnement tot het maken van aangepaste beleidsregels die de ritten definiëren die uw klanten aangaan. |
| [Partners, leveranciers en klanten toegang geven tot toepassingen met Active Directory van Azure](https://aka.ms/aadexternalidentities) | Het succes van elke organisatie, ongeacht de omvang, branche of naleving en beveiligingshouding, is afhankelijk van het organisatorische vermogen om samen te werken met andere organisaties en in contact te komen met klanten.<br><br>Met Azure AD, Azure AD B2C en Azure AD B2B-samenwerking worden in deze handleiding de bedrijfswaarde en de mechanismen van het bouwen van een toepassing of webervaring beschreven die een geconsolideerde verificatie-ervaring biedt die is afgestemd op de contexten van uw werknemers, zakenpartners en leveranciers en klanten. |
| [Toepassingsverificatie migreren naar Azure AD B2C in een hybride omgeving](https://aka.ms/MigratetoAADB2C) | In de moderne organisaties van vandaag, digitale transformatie en de overgang naar de cloud gebeurt in fasen, waarbij de meeste organisaties op zijn minst tijdelijk opereren in een hybride identiteit omgeving.<br><br>Deze handleiding richt zich op het maken van het migratieplan voor het verplaatsen van uw eerste toepassing naar Azure AD B2C en behandelt de overwegingen om dit te doen terwijl u zich in een hybride identiteitsomgeving bevindt. |
| [Algemene Verordening Gegevensbescherming (AVG) Overwegingen voor klantgerichte toepassingen](https://aka.ms/AADB2CandGDPR) | Voor elke klant die met toepassingen te maken heeft, moet gdpr in aanmerking worden genomen door alle organisaties die projecten uitvoeren die persoonsgegevens bevatten en EU-burgers dienen.<br><br>Deze oplossingshandleiding richt zich op hoe Azure AD B2C kan worden gebruikt als een flexibel onderdeel van uw algemene GDPR-nalevingsaanpak, inclusief hoe Azure AD B2C-componenten elk van de belangrijkste GDPR-rechten voor individuen ondersteunen. |
