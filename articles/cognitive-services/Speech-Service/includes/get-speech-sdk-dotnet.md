#### <a name="c-nuget-package"></a>C# NuGet-pakket

De .NET Speech SDK kan met de volgende `dotnet add` opdracht worden geïnstalleerd vanaf de **.NET Core CLI.**

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

De .NET Speech SDK kan **Package Manager** met de `Install-Package` volgende opdracht vanuit Package Manager worden geïnstalleerd.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
