# <a name="get-started-with-the-mailchimp-connector"></a>Aan de slag met de MailChimp-connector
MailChimp is een SaaS-service waarmee bedrijven, beheren en automatiseren van e-mailmarketingactiviteiten, zoals het verzenden van marketing-e-mails, geautomatiseerde berichten en gerichte campagnes.

U kunt aan de slag met het maken van een logische app nu, Zie [maken van een logische app](../logic-apps/quickstart-create-first-logic-app-workflow.md).

## <a name="create-a-connection-to-mailchimp"></a>Maak een verbinding met MailChimp
Als u wilt maken van logische apps met MailChimp, moet u eerst maken een **verbinding** geeft u de details voor de volgende eigenschappen:

| Eigenschap | Vereist | Description |
| --- | --- | --- |
| Token |Ja |MailChimp-referenties opgeven |
