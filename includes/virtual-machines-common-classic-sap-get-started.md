## <a name="planning-and-implementation"></a>Planning en implementatie
Titel: SAP NetWeaver on Azure Virtual Machines – Planning and Implementation Guide

Samenvatting: dit is het document waarmee u begint als u overweegt SAP NetWeaver te gebruiken op virtuele Azure-machines. Op basis van deze plannings- en implementatiehandleiding kunt u bepalen of een bestaand of gepland SAP NetWeaver-systeem kan worden geïmplementeerd in een omgeving met virtuele Azure-machines. Hierin komen verschillende SAP NetWeaver-implementatiescenario’s aan bod en worden specifieke SAP-configuraties voor Azure beschreven. Het document bevat alle configuratiegegevens die u voor SAP/Azure nodig hebt om een hybride SAP-landschap uit te voeren. Daarnaast wordt beschreven welke maatregelen u kunt nemen om een hoge beschikbaarheid van SAP NetWeaver-systemen in IaaS te garanderen.

Bijgewerkt: augustus 2015

## <a name="sap-dbms-on-azure"></a>SAP DBMS op Azure
Titel: SAP DBMS in Azure Deployment Guide

Samenvatting: dit document bevat plannings- en implementatie-informatie voor de DBMS-systemen die moeten worden uitgevoerd in combinatie met SAP. Het eerste deel bestaat uit algemene overwegingen. De volgende delen van het artikel hebben betrekking op implementaties van verschillende DBMS-systemen in Azure die worden ondersteund door SAP. Verschillende DBMS gepresenteerd zijn SQL Server, SAP ASE, Oracle, SAP MaxDB en IBM DB2 voor Linux, Unix en Windows. Er wordt aangegeven waarmee u rekening moet houden wanneer u SAP-systemen in Azure uitvoert in combinatie met deze DBMS-systemen. Er wordt onder andere beschreven welke back-ups en methoden voor hoge beschikbaarheid door de verschillende DBMS-systemen in Azure worden ondersteund voor gebruik met SAP-toepassingen.

Bijgewerkt: december 2015

## <a name="sap-netweaver-on-azure"></a>SAP NetWeaver op Azure
Titel: SAP NetWeaver - Building an Azure based Disaster Recovery Solution

Samenvatting:Dit document biedt stapsgewijze richtlijnen voor het bouwen van een azure-gebaseerde disaster recovery-oplossing voor SAP NetWeaver. De beschreven oplossing gaat ervan uit dat het SAP-landschap wordt uitgevoerd gevirtualiseerd on-premises op basis van Hyper-V. In het eerste deel van het document worden ASR-services (Document Azure Site Recovery) geïntroduceerd in de componenten. Het tweede deel van het document beschrijft details voor op SAP NetWeaver gebaseerde landschappen. Mogelijkheden om ASR te gebruiken met SAP NetWeaver applicatie-exemplaren en SAP Central Services worden besproken en beschreven. Een focus van het tweede deel is het gebruik van ASR voor SAP Central Services die zijn beveiligd met windows Server Failover Cluster-configuraties.

Bijgewerkt: september 2015
