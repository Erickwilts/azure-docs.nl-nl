## <a name="create-an-azure-resource"></a>Een Azure-resource maken

Begin met het gebruik van de Bing Afbeeldingen zoeken-API door een van de volgende Azure-resources te maken.

### <a name="bing-search-v7-resource"></a>[Bing Search V7-resource](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesBingSearch-v7)
   * Beschikbaar via de Azure Portal totdat u de resource verwijdert.
   * Gebruik de gratis prijs categorie om de service te proberen en pas later bij te werken naar een betaalde laag voor productie.

### <a name="multi-service-resource"></a>[Resource met meerdere services](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAllInOne)
   * Beschikbaar via de Azure Portal totdat u de resource verwijdert.  
   * Gebruik dezelfde sleutel en hetzelfde eind punt voor uw toepassingen, op meerdere Cognitive Services.
