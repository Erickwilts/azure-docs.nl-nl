Voeg uw marketing-specifieke inhoud toe aan de sectie **overzicht** .

## <a name="overview"></a>Overzicht

In deze sectie voert u de algemene informatie in over uw Azure Marketplace-aanbieding.  Een asterisk (*) die wordt toegevoegd aan de veld naam geeft aan dat deze vereist is.

![Marketplace-informatie](./media/containers-marketplace-tab.png)

In de volgende tabel worden het doel en de inhoud van deze velden beschreven. Vereiste velden worden gedicteerd door een asterisk (*).

|    Veld                  |       Beschrijving                                                            |
|  ---------                |     ---------------                                                          |
| **Titel\***                 | De titel van de aanbieding. Deze wordt duidelijk weer gegeven in de Marketplace.  De maximale lengte is 50 tekens. |
| **Samenvatting\***               | Korte samen vatting van de aanbieding. De maximale lengte is 100 tekens. |
| **Lange samen vatting\***          | Een langere samen vatting van de aanbieding (hoewel deze mogelijk hetzelfde is als de **samen vatting**).  De maximale lengte is 256 tekens. |
| **Beschrijving\***           | De beschrijving van de aanbieding.  De maximale lengte is 3000 tekens en biedt ondersteuning voor eenvoudige HTML-opmaak.
| **Marketing-id**  | Een unieke URL om te koppelen aan deze aanbieding, die doorgaans uw organisatie en naam van de oplossing bevat. De maximale lengte is 50 tekens.   |
| **Preview-abonnement-Id's\*** | Voeg een id toe aan 100-abonnement-id's van Previewers. Deze op wit vermelde abonnementen hebben toegang tot de aanbieding nadat deze is gepubliceerd, voordat ze live gaan. |
|  |  |
