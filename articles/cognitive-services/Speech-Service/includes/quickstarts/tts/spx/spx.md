## <a name="run-the-speech-cli"></a>De speech CLI uitvoeren

U bent nu klaar om de speech CLI uit te voeren om spraak vanuit tekst te kunnen gebruiken.

Ga vanaf de opdracht regel naar de map die het binaire bestand speech CLI bevat en typ het volgende:

```bash
spx synthesize --text "The speech synthesizer greets you!"
```

De spraak-CLI produceert de natuurlijke taal in het Engels via de computer luidspreker.
