De volgende limieten gelden voor Azure Backup.

| **Limiet** | **Standaard** |
| --- | --- |
| Servers of computers die kunnen worden geregistreerd in een kluis. | Windows Server/Windows-client/System Center Data Protection Manager: 50. <br/><br/> IaaS Vm's: 1.000.  |
| Grootte van een gegevens bron in de kluis opslag. |Maxi maal 54.400 GB. De limiet is niet van toepassing op back-ups van IaaS-VM'S. |
| Back-upkluizen in een Azure-abonnement. |500 kluizen per regio. |
| Dagelijkse back-ups plannen. |Windows-Server/-client: Drie per dag.<br/> System Center DPM: Twee dagen. <br/> IaaS Vm's: Eenmaal per dag.  |
| Gegevens schijven die zijn gekoppeld aan een Azure-VM voor back-up. | 16 |
| De afzonderlijke gegevens schijf die is gekoppeld aan de Azure VM voor back-up.| 4\.095 GB|
