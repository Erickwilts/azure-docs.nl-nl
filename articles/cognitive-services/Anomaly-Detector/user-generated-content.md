# <a name="featured-user-generated-content-for-the-anomaly-detector-api"></a>Aanbevolen door de gebruiker gegenereerde inhoud voor de Api voor anomaliedetector

Gebruik dit artikel om te ontdekken hoe andere klanten denken over en het gebruik van de Anomalie Detector API. De volgende bronnen zijn gemaakt door de gemeenschap van Anomaly Detector-gebruikers. Ze omvatten open-source projecten, en andere bijdragen gemaakt door zowel Microsoft en derden gebruikers. Sommige van de volgende links worden gehost op websites die buiten Microsoft staan en Microsoft is daar niet verantwoordelijk voor de inhoud. Gebruik discretie wanneer u naar deze bronnen verwijst.
