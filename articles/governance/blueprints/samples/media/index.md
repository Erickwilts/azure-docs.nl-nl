# <a name="overview-of-the-media-blueprint-sample"></a>Overzicht van het blauwdrukvoorbeeld voor Media

Het blauwdrukvoorbeeld voor Media biedt een set beheerrichtlijnen op basis van [Azure Policy](../../../policy/overview.md) die bijdragen aan de [Media](https://www.hhs.gov/hipaa/for-professionals/security/laws-regulations/index.html)-attestatie. 

## <a name="blueprint-sample"></a>Blauwdrukvoorbeeld

Met behulp van deze blauwdrukvoorbeelden kunnen klanten een kernset beleidsregels implementeren voor elke Azure-architectuur waarvoor accreditatie of naleving van het Media-framework is vereist. De sectie over [toewijzing van beheeropties](./control-mapping.md) biedt meer informatie over de beleidsregels die in dit initiatief zijn opgenomen en hoe dit beleid kan bijdragen aan verschillende beheeropties die zijn gedefinieerd in het Media-framework.
