# <a name="create-an-azure-application-offer"></a>Een Azure-toepassings aanbieding maken

In dit artikel wordt beschreven hoe u een vermelding voor een Azure-toepassings aanbieding maakt en publiceert voor Azure Marketplace. Elk aanbod wordt als eigen entiteit weer gegeven in azure Marketplace en bestaat uit de volgende groeperingen van assets en ondersteunende services: Details van de aanbieding, Sku's, marketing artefacten en ondersteunings informatie.

## <a name="new-offer-form"></a>Formulier voor nieuwe aanbieding

Meld u aan bij de [Cloud Partner-Portal](https://cloudpartner.azure.com/)en selecteer **+ nieuwe aanbieding** op de menu balk aan de linkerkant. Selecteer in het menu nieuwe aanbieding de optie **Azure-toepassingen** om het **nieuwe aanbod** formulier weer te geven en start het proces van het definiÃ«ren van assets voor een nieuwe aanbieding.
