# <a name="sign-in-activity-report-error-codes"></a>Foutcodes van aanmeldingsactiviteitsrapport 

Met de informatie die door het [gebruikersaanmeldingsrapport](concept-sign-ins.md)wordt verstrekt, vindt u antwoorden op vragen zoals:

- Wie heeft zich aangemeld bij mijn aanvraag?
- Bij welke aanvragen zijn aangemeld?
- Welke aanmeldingen mislukten en waarom?

Wanneer een aanmelding mislukt, ziet u een foutcode die overeenkomt met de fout. [Zoek de beschrijving voor een foutcode](https://login.microsoftonline.com/error).
