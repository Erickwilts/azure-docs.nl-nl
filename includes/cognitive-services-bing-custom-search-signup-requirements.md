## <a name="create-an-azure-resource"></a>Een Azure-resource maken

Begin met het gebruik van de Bing Custom Search-API door een van de volgende Azure-resources te maken.

### <a name="trial-resource"></a>[Proef resource](https://azure.microsoft.com/try/cognitive-services/?api=bing-custom-search-api)
   * Er is geen Azure-abonnement nodig.
   * Gratis geldig voor zeven dagen. Nadat u zich hebt aangemeld, zijn er een proef-en-eind punt beschikbaar op de [Azure-website](https://azure.microsoft.com/try/cognitive-services/my-apis/).

### <a name="bing-custom-search-resource"></a>[Bing Aangepaste zoekopdrachten resource](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesBingCustomSearch)
   * Beschikbaar via de Azure Portal totdat u de resource verwijdert.
   * Gebruik de gratis prijs categorie om de service te proberen en pas later bij te werken naar een betaalde laag voor productie.

### <a name="multi-service-resource"></a>[Resource met meerdere services](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAllInOne)
   * Beschikbaar via de Azure Portal totdat u de resource verwijdert.  
   * Gebruik dezelfde sleutel en hetzelfde eind punt voor uw toepassingen, op meerdere Cognitive Services.
