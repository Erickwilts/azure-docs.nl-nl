#### <a name="to-install-regular-updates-via-the-azure-classic-portal"></a>Regelmatige updates installeren via de klassieke Azure-portal
1. Selecteer **op** de pagina Apparaten het apparaat waarop u updates wilt installeren.
2. Navigeer naar **Onderhoud van apparaten** > **Maintenance** en blader omlaag naar **Software-updates.**
3. Als u wilt controleren op updates, klikt u onder aan de pagina op **Updates controleren.**
4. U ziet een bericht als er software-updates beschikbaar zijn. Klik **op Updates installeren** om het apparaat bij te werken.
