Als u de cmdlets wilt uitvoeren, kunt u Azure Cloud Shell gebruiken, een gratis interactieve shell. In deze shell zijn algemene Azure-hulpprogramma's vooraf geïnstalleerd en geconfigureerd voor gebruik met uw account. Klik op **Kopiëren** om de code te kopiëren, plak deze in Cloud Shell en druk vervolgens op Enter om de code uit te voeren. U kunt Cloud Shell op verschillende manieren starten:


|  |   |
|-----------------------------------------------|---|
| Open Cloud Shell in uw browser. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Klik op de knop **Cloud shell** in het menu rechtsboven in de portal. | [![Cloud Shell in de portal](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Als u Azure Cloud shell niet wilt gebruiken, kunt u in plaats daarvan lokaal Power Shell installeren. Als u Power shell lokaal wilt installeren en gebruiken, moet u ervoor zorgen dat u de meest recente versie van de Power shell-cmdlets voor Resource Manager installeert. Power shell-cmdlets worden regel matig bijgewerkt en u moet normaal gesp roken uw Power shell-cmdlets bijwerken om de nieuwste functie functionaliteit te krijgen, waardoor u mogelijk problemen ondervindt.

Als u de versie van Power shell wilt vinden die u lokaal uitvoert, gebruikt u de cmdlet Get-module-ListAvailable AZ. Zie [de module Azure PowerShell installeren](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps)om bij te werken. Zie [Azure PowerShell installeren en configureren](https://docs.microsoft.com/powershell/azure/azurerm/overview) voor meer informatie.

Als u Power shell gebruikt in macOS, volgt u de stappen in [Power shell Core installeren op MacOS.](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6)
