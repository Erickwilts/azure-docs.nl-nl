# <a name="saas-fulfillment-apis"></a>SaaS-uitvoerings-API's

De SaaS Fulfillment API's stellen onafhankelijke softwareleveranciers (ISV's) in staat om hun SaaS-toepassingen te integreren met de Azure Marketplace. Deze API's stellen ISV-toepassingen in staat om deel te nemen aan alle commerce-kanalen: direct, partner-led (reseller) en field-led.  Ze zijn een vereiste voor het aanbieden van transactiesde SaaS-aanbiedingen op de Azure Marketplace.

> [!WARNING]
> De huidige versie van deze API is versie 2, die moet worden gebruikt voor alle nieuwe SaaS-aanbiedingen.  Versie 1 van de API wordt afgeschaft en wordt onderhouden om bestaande aanbiedingen te ondersteunen.


## <a name="business-model-support"></a>Ondersteuning voor bedrijfsmodel

Deze API ondersteunt de volgende mogelijkheden voor bedrijfsmodelen; U kunt:

* Geef meerdere plannen op voor een aanbieding. Deze plannen hebben verschillende functionaliteit en kunnen anders worden geprijsd.
* Bied een aanbieding op basis van een per site of per gebruiker.
* Geef maandelijkse en jaarlijkse (betaalde vooraf) factureringsopties.
