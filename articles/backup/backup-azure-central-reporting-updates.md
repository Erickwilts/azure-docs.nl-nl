# <a name="update-the-azure-backup-central-reporting-content-pack"></a>Het inhouds pakket voor de Azure Backup Central-rapportage bijwerken 

Het [Azure backup inhouds pakket](https://docs.microsoft.com/azure/backup/backup-azure-configure-reports#view-reports-in-power-bi) kan worden gebruikt om rapporten over centrale back-ups weer te geven. Het inhouds pakket wordt regel matig bijgewerkt om meer functies toe te voegen en fouten op te lossen. In dit artikel leest u hoe u het inhouds pakket bijwerkt. Ook wordt uitgelegd hoe u de update uitstelt en hoe de updates na verloop van tijd worden uitgevoerd.

## <a name="get-updates-to-the-content-pack"></a>Updates voor het inhouds pakket ophalen

### <a name="get-the-updated-content-pack"></a>Het bijgewerkte inhouds pakket ophalen
Als u geen wijzigingen hebt aangebracht in uw exemplaar van het inhouds pakket, wordt het automatisch bijgewerkt. Wanneer het inhouds pakket wordt gewijzigd, ontvangt u een melding in Power BI en een e-mail melding. U kunt ervoor kiezen om het bijgewerkte inhouds pakket op uw gemak te verkrijgen. 

### <a name="postpone-the-update"></a>De update uitstellen
De best practice is het importeren van het inhouds pakket in een [aangepaste werk ruimte](https://youtu.be/26zyOtyHPJM?t=1m57s). U kunt de rapporten nu bewerken.
Zoals eerder vermeld, ziet u een melding in Power BI als het inhouds pakket wordt gewijzigd. U kunt ervoor kiezen om het inhouds pakket later op te halen. 

## <a name="coming-soon"></a>Binnenkort beschikbaar
   
Het Azure Backup inhouds pakket is bijgewerkt om meer werk belastingen te ondersteunen. Werk belastingen bevatten Azure SQL Database voor IaaS VM-back-ups en System Center Data Protection Manager. Deze ondersteuning wordt toegevoegd aan de huidige ondersteuning voor Azure backup-en Azure VM-back-ups. Deze ondersteuning betekent dat u al uw back-upgegevens op één centrale locatie kunt weer geven en analyseren. [Rapporten kunnen ook worden aangepast](https://youtu.be/26zyOtyHPJM) aan de behoeften van uw organisatie.

De vooraf geconfigureerde rapporten die worden geleverd bij het Azure Backup inhouds pakket, worden gewijzigd. De wijzigingen maken de rapporten duidelijker in werk belastingen. U vindt hier een alvast-kijkje in de aanstaande set met rapporten.
