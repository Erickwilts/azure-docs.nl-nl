# <a name="storsimple-for-cloud-solutions-provider-program-frequently-asked-questions"></a>StorSimple for Cloud Solutions Provider Program: veelgestelde vragen


## <a name="overview"></a>Overzicht
Hieronder vindt u vragen en antwoorden die u als StorSimple-partner zou kunnen hebben wanneer u een StorSimple Virtual Array implementeert of beheert in Azure-portal.

Vragen en antwoorden zijn ingedeeld in de volgende categorieën:

 - [StorSimple en Cloud Solutions Provider Program (CSP) model](#storsimple-and-csp-model)
 - [Virtuele array implementeren en beheren als partner](#deploy-and-manage-storsimple-as-a-partner)
 - Ondersteuning voor StorSimple partners voor CSP

## <a name="storsimple-and-csp-model"></a>StorSimple en CSP-model

### <a name="what-does-it-mean-to-have-storsimple-as-a-part-of-the-cloud-solutions-provider-csp-program"></a>Wat betekent het om StorSimple te hebben als onderdeel van het Cloud Solutions Provider (CSP) programma?
 StorSimple voor CSP stelt onze CSP-partners in staat om StorSimple Virtual Array hun klanten door te verkopen. Dit model stelt de CSP-partners in staat om de end-to-end levenscyclus van klanten te bezitten met directe inrichting, facturering en ondersteuning van de cloudservices van Microsoft. 

Voor meer informatie, ga naar [StorSimple in CSP-programma.](https://www.microsoft.com/en-us/server-cloud/products/storsimple/) 

### <a name="is-storsimple-a-usage-based-or-based-on-seat-licenses-like-office-365-and-enterprise-mobility-suite-ems"></a>Is StorSimple een gebruiksgebaseerd eis of gebaseerd op stoellicenties zoals Office 365 en Enterprise Mobility Suite (EMS)?
StorSimple is een op gebruik gebaseerde service. Klanten die StorSimple Virtual Array via de CSP-route aanschaffen, betalen voor het gebruik van virtual array en Azure, gefactureerd als afzonderlijke regelitems in de CSP-factuur. 

### <a name="is-the-storsimple-business-model-similar-to-that-of-other-services-such-as-the-office-365-and-the-enterprise-mobility-suite"></a>Is het StorSimple-bedrijfsmodel vergelijkbaar met dat van andere services zoals de Office 365 en de Enterprise Mobility Suite? 
Ja. Het model is een wholesale-kortingsmodel, net als andere op gebruik gebaseerde services in Azure. De wholesalekorting voor StorSimple is vergelijkbaar met die van andere Azure-services. Dit model biedt onze CSP-partners de mogelijkheid om meer te verkopen en Azure-verbruik te stimuleren via StorSimple Virtual Array.  

### <a name="which-storsimple-skus-are-available-at-launch"></a>Welke StorSimple SKU's zijn beschikbaar bij de lancering?
Dezelfde SKU's die beschikbaar zijn met de standalone StorSimple-services. 

### <a name="is-there-any-difference-between-the-storsimple-directly-purchased-from-microsoft-under-ea-and-storsimple-as-a-part-of-csp"></a>Is er een verschil tussen de StorSimple rechtstreeks gekocht van Microsoft onder EA en StorSimple als onderdeel van CSP?
Nee. Onder CSP kunnen klanten ook andere diensten kopen die door de CSP-partner worden aangeboden, samen met StorSimple Virtual Array (SVA) onder één factuur. Houd er rekening mee dat voor de aanschaf van de StorSimple 8000-serie fysieke apparaat, klanten nog steeds moeten gaan via de EA-route. 

### <a name="in-which-markets-is-storsimple-available-for-csp-at-launch"></a>In welke markten, is StorSimple beschikbaar voor CSP bij de lancering?
StorSimple Virtual Array voor CSP is beschikbaar in alle markten waar StorSimple vandaag beschikbaar is. Ga voor meer informatie naar de [lijst met regio's waar StorSimple beschikbaar is.](https://azure.microsoft.com/regions/services/)

### <a name="what-kind-of-solutions-can-a-partner-deliver-with-storsimple-and-csp"></a>Welke oplossingen kan een partner leveren met StorSimple en CSP? 
Er zijn meerdere oplossingen:

- De partners kunnen StorSimple Virtual Array doorverkopen. 
- De partners kunnen oplossingen leveren die zijn gebouwd rond StorSimple en andere Azure-services of software van derden. Voor meer informatie, bekijk een aantal mogelijke [oplossingen die bestaan rond StorSimple](https://aka.ms/storsimple-build-solutions).
- Ze kunnen ook nieuwe bedrijfsmodellen ontdekken door managed services te leveren.

### <a name="what-are-the-incentives-available-for-storsimple-csp-partners-do-i-qualify-for-any-csp-program-incentives"></a>Wat zijn de prikkels beschikbaar voor StorSimple CSP partners? Kom ik in aanmerking voor een CSP-programma incentives?
Voor informatie over incentives, ga naar [CSP programma incentives](https://partner.microsoft.com/en-US/cloud-solution-provider/program-details).


## <a name="deploy-and-manage-storsimple-as-a-partner"></a>StorSimple implementeren en beheren als partner

### <a name="how-can-i-administer-storsimple-virtual-array-in-csp"></a>Hoe kan ik StorSimple Virtual Array beheren in CSP? 
U StorSimple Virtual Array-abonnementen toevoegen aan het account van uw klanten via het Partnercenter. Bovendien u de Azure-portal gebruiken om gebruikers aan de abonnementen toe te voegen.

### <a name="is-the-azure-portal-approach-the-same-as-other-services-for-csp"></a>Is de Azure-portalbenadering hetzelfde als andere services voor CSP? 
Ja. Met StorSimple Virtual Array voor CSP is de beste manier voor partners om toegang te krijgen tot de Azure-portal via het [Partnercenter,](https://partnercenter.microsoft.com/) waar ze klanten en abonnementen kunnen beheren. Als u StorSimple-abonnementen wilt beheren, moet de partner of klant (afhankelijk van de verleende machtigingen) zich aanmelden bij de Azure-portal. 

### <a name="is-microsoft-shipping-a-new-portal-for-storsimple-for-csp"></a>Is Microsoft de scheepvaart een nieuwe portal voor StorSimple voor CSP? 
Nee. U StorSimple Virtual Array beheren in CSP via de Azure-portal.

### <a name="i-have-provisioned-a-standalone-storsimple-subscription-for-my-customer-do-i-have-to-use-the-azure-portal-to-administer-my-device"></a>Ik heb een standalone StorSimple abonnement voor mijn klant. Moet ik de Azure-portal gebruiken om mijn apparaat te beheren? 
Ja. 

### <a name="is-the-microsoft-account-team-compensated-as-a-result-of-sales-of-storsimple-through-csp"></a>Wordt het Microsoft-accountteam gecompenseerd als gevolg van de verkoop van StorSimple via CSP?
Ja. De partner sales executive en het klantenaccount team voor de eindklant worden gecompenseerd voor CSP sales. Van de partners wordt verwacht dat zij het voortouw nemen bij de verkoop van hun gedifferentieerde waarde en aanbod.
