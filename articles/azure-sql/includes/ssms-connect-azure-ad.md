> [!NOTE]
> In december 2021 kunnen releases van SSMS vóór 18.6 niet meer worden geverifieerd via Azure Active Directory met MFA.
>
> Als u wilt doorgaan met het gebruik van Azure Active Directory-verificatie met MFA, hebt u [SSMS 18.6 of hoger](https://docs.microsoft.com/sql/ssms/download-sql-server-management-studio-ssms) nodig.
