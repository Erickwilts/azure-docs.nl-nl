# <a name="what-is-microsoft-genomics"></a>Wat is Microsoft Genomics?
Microsoft Genomics biedt een cloudimplementatie van de Burrows-Wheeler Aligner (BWA) en de Genome Analysis Toolkit (GATK) voor secundaire analyse. De service is ISO-gecertificeerd en voldoet aan de HIPAA-regelgeving en biedt prijsvoorspelbaarheid voor uw genoomsequencingbehoeften. Lees hoe u de Microsoft Genomics-service gebruiken en integreren met onze API door onze quickstarts, tutorials en documentatie te lezen.

## <a name="support-your-most-demanding-sequencing-needs"></a>Ondersteun uw meest veeleisende sequencing-behoeften
In plaats van uw eigen datacenters te beheren, u profiteren van de schaal en ervaring van Microsoft bij het uitvoeren van workloads op exabyte-schaal. Onze cloudimplementatie van de BWA-GATK is zeer in overeenstemming met de best practices pipeline van het Broad Institute.


## <a name="keep-your-business-running"></a>Houd uw bedrijf draaiende
Microsoft Genomics biedt een SLA (99,99% beschikbaarheidsserviceniveauovereenkomst) voor het ontvangen van werkstroomaanvragen. Meer informatie over de Microsoft Genomics [SLA](https://azure.microsoft.com/support/legal/sla/genomics/v1_0/).


## <a name="secure-your-data"></a>Uw gegevens beveiligen
De Microsoft Genomics-service is ISO 27001, ISO 27018 en ISO 9001 gecertificeerd en voldoet aan de HIPAA-regelgeving. Bezoek het [Vertrouwenscentrum van Azure](https://www.microsoft.com/trustcenter/security) voor informatie over de beveiliging van het Azure-platform.
