# <a name="get-started-with-built-in-iot-hub-integration"></a>Aan de slag met ingebouwde IoT Hub-integratie

Met deze optie u de service gebruiken zonder Azure Security Center voor IoT-beveiligingsagents te gebruiken.

## <a name="enable-built-in-iot-hub-integration"></a>Ingebouwde IoT Hub-integratie inschakelen

Ga als volgt te werk om het beheer van uw apparaat, het beheer van apparaat naar cloud en communicatievan cloud naar apparaat te controleren:

1. Open uw **IoT-hub.**
1. Selecteer het **menu Beveiligingsoverzicht.**
1. Klik **op Uw IoT-oplossing beveiligen** en vul het onboarding-formulier in.

Gefeliciteerd! U hebt de Azure Security Center for IoT-service ingeschakeld op uw IoT-hub.
