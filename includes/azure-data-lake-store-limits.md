Azure Data Lake Storage Gen1 is een bedrijfsbrede opslag ruimte voor big data analytische werk belastingen. U kunt Data Lake Storage Gen1 gebruiken voor het vastleggen van gegevens van elke grootte, type en opname snelheid op één plek voor operationele en experimentele analyses. Er is geen limiet voor de hoeveelheid gegevens die u kunt opslaan in een Data Lake Storage Gen1-account.

| **Resource** | **Standaardlimiet** | **opmerkingen** |
| --- | --- | --- |
| Maximum aantal Data Lake Storage Gen1 accounts, per abonnement, per regio |10 | Neem contact op met de ondersteuning om een verhoging van deze limiet aan te vragen. |
| Maximum aantal toegangs-Acl's, per bestand of map |32 | Dit is een vaste limiet. Gebruik groepen om de toegang te beheren met minder vermeldingen. |
| Maximum aantal standaard-Acl's, per bestand of map |32 | Dit is een vaste limiet. Gebruik groepen om de toegang te beheren met minder vermeldingen. |
