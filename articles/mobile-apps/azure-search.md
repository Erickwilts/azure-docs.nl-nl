# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>AI-aangedreven cloudservice met Azure Cognitive Search
[Azure Cognitive Search](https://azure.microsoft.com/services/search/) is een search-as-a-service cloudoplossing die ontwikkelaars API's en tools biedt voor het toevoegen van een rijke zoekervaring over priv√©, heterogene inhoud in web-, mobiele en bedrijfstoepassingen. Uw aangepaste code roept gegevensopname (indexering) op om een index te maken en te laden. Aan de andere kant geeft uw toepassingscode queryaanvragen op en verwerkt het antwoorden. De zoekervaring wordt in uw client gedefinieerd met behulp van functionaliteit van Azure Cognitive Search, met query-uitvoering over een uitgebreide index die u in uw service maakt, bezit en opslaat.

## <a name="azure-cognitive-search-features"></a>Azure Cognitive Search-functies
- **Zoeken naar tekst in vrije vorm:** biedt zoeken in volledige tekst, eenvoudige syntaxis van query's en de syntaxis van Lucene-query.
- **Eenvoudige scoreprofielen**: Modelrelevantie als functie van waarden in de documenten zelf.
- **Geosearch**: verkent gegevens op basis van de nabijheid van een zoekresultaat naar een fysieke locatie.
- **Functies voor gebruikerservaringen:** geef automatisch aanvullen, zoeksuggesties, sorteren en paging.
- **Cognitief zoeken:** kan worden toegepast op een indexeringspijplijn om tekstinformatie uit ruwe inhoud te extraheren voor beeld- en tekstanalyse.
- **Kennisarchief**: Bespaart verrijkingen die tijdens het indexeren zijn gemaakt.
- **Gegevensbronnen:** haal doorzoekbare inhoud uit in primaire gegevensarchieven wanneer u bent verbonden met Azure SQL Database, Azure Cosmos DB en Azure Blob-opslag.
- **Wizard Gegevens importeren:** hiermee configureert u indexen. 
- **Indexontwerper**: Staat een index omhoog. 
- **Zoekverkenner:** hiermee worden query's getest en scoreprofielen verfijnd.
