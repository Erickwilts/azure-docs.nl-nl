# <a name="download-notebookspreview-data"></a>Notitie blokken downloaden (preview)-gegevens

De functie notebooks (preview) wordt verwijderd op *13 2020 april*. Na die datum verdwijnt het tabblad notebooks (preview) en kunnen de notitieblok gegevens niet worden hersteld. Down load de gegevens van uw notitie blokken vóór 13 2020 april.

In dit artikel vindt u stapsgewijze instructies voor het downloaden van notitie blokken (preview-gegevens).

1. Ga naar het tabblad **notebooks** in azure machine learning Studio (klassiek).
    ![gaat u naar het tabblad notebooks.](./media/download-notebooks/notebooks-list.png)

1. Selecteer het notitie blok dat u wilt downloaden. Deze wordt geopend in Jupyter.
    ![een notitie Blok selecteren](./media/download-notebooks/select-notebook.png) het downloaden van meerdere notitie blokken tegelijk niet wordt ondersteund.

1. Ga naar **bestanden** -> **down load als**en selecteer een optie voor de indeling.
    opties voor downloaden van ![notitie blok](./media/download-notebooks/download-options.PNG)

1. Nadat u een optie hebt geselecteerd, wordt het notebook-bestand gedownload. Alle 500-of 502-fouten negeren die worden weer gegeven, zijn niet van invloed op uw down load.
