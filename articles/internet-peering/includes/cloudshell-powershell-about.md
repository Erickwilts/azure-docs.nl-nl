Als u de cmdlets wilt uitvoeren, u Azure Cloud Shell gebruiken, een gratis interactieve shell. In deze shell zijn algemene Azure-hulpprogramma's vooraf geïnstalleerd en geconfigureerd voor gebruik met uw account. Selecteer **Kopiëren** om de code te kopiëren en plak deze in Cloud Shell. Selecteer vervolgens **Enter** om het uit te voeren. Er zijn een paar manieren om Cloud Shell te lanceren:


|  |   |
|-----------------------------------------------|---|
| Open Cloud Shell in uw browser. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Selecteer de knop **Cloud Shell** op de werkbalk rechtsboven in de Azure-portal. | [![Cloud Shell in de portal](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Als u Azure Cloud Shell niet wilt gebruiken, u PowerShell lokaal installeren. Als u ervoor kiest PowerShell lokaal te installeren en te gebruiken, moet u de nieuwste versie van de PowerShell-cmdlets van Azure Resource Manager installeren. PowerShell-cmdlets worden regelmatig bijgewerkt. U moet uw PowerShell-cmdlets doorgaans bijwerken om de nieuwste functiefunctionaliteit te krijgen. Als u dat niet doet, u problemen ondervinden.

Als u de versie van PowerShell wilt vinden die u lokaal uitvoert, gebruikt u de cmdlet **'Get-Module -ListAvailable Az'.** Zie De [Azure PowerShell-module installeren](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps)als u deze wilt bijwerken. Zie voor meer informatie hoe u [Azure PowerShell installeert en configureert.](https://docs.microsoft.com/powershell/azure/azurerm/overview)

Als u PowerShell op macOS gebruikt, volgt u de stappen voor [het installeren van PowerShell op macOS.](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6)
