# <a name="get-started-with-azure-synapse-analytics"></a>Aan de slag met Azure Synapse Analytics

Deze zelfstudie is een stapsgewijze gids door de belangrijkste functiegebieden van Azure Synapse Analytics. De zelfstudie is het ideale startpunt voor iemand die een rondleiding wil door de belangrijkste scenario's van Azure Synapse Analytics. Nadat u de stappen in de zelfstudie hebt uitgevoerd, hebt u een volledig functionele Synapse-werkruimte waarin u gegevens kunt analyseren met behulp van SQL, SQL on demand en Apache Spark.

U leert:
* Een Synapse-werkruimte inrichten in een Azure-abonnement
* Toegangsbeheer configureren voor een ADLSGEN2-account zodat dit naadloos werkt met de Synapse-werkruimte
* De NYCTaxi-voorbeeldgegevens laden in de Synapse-werkruimte zodat deze kunnen worden gebruikt door SQL, SQL on demand en Spark
* SQL-scripts en Synapse-notebooks bewerken en uitvoeren met behulp van Synapse Studio
* Query's uitvoeren op SQL-tabellen en Spark-tabellen
* Gegevens uit SQL-tabellen in Spark-dataframes laden
* Gegevens uit Spark-dataframes in SQL-tabellen laden
* De inhoud van een ADLSGEN2-account verkennen
* Parquet-gegevensbestanden in ADLSGEN2-accounts met behulp van Spark en SQL on demand 
* Een gegevenspijplijn maken die elk uur automatisch een Synapse-notebook uitvoert
