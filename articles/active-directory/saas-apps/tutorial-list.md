# <a name="tutorials-for-integrating-saas-applications-with-azure-active-directory"></a>Zelf studies voor het integreren van SaaS-toepassingen met Azure Active Directory

We hebben een verzameling zelf studies ontwikkeld waarmee u de configuratie kunt door lopen om uw SaaS-toepassingen [(software als een service)](https://azure.microsoft.com/overview/what-is-saas/) in de cloud te integreren met Azure Active Directory.

Zie de [Active Directory Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps/category/azure-active-directory-apps)voor een lijst met alle SaaS-apps die vooraf zijn geïntegreerd in azure AD. Klik [hier](https://docs.microsoft.com/azure/active-directory/develop/howto-app-gallery-listing) om een app aan te vragen die aan de toepassings galerie moet worden toegevoegd.
