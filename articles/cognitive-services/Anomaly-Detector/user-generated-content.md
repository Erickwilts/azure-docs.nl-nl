# <a name="featured-user-generated-content-for-the-anomaly-detector-api"></a>Aanbevolen door gebruikers gegenereerde inhoud voor de API van de detectie van afwijkingen

Gebruik dit artikel om te ontdekken hoe andere klanten nadenken over en met behulp van de detectie van afwijkingen API. De volgende resources zijn gemaakt door de community van gebruikers van de detectie van afwijkingen.
