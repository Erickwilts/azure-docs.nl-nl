# <a name="users-page"></a>Pagina voor Gebruikers

Op de pagina [**gebruikers**](https://cloudpartner.azure.com/#users) wordt een lijst weer gegeven met alle geregistreerde gebruikers van de Cloud Partner-Portal.  De kolommen bevatten het e-mail adres van de gebruiker, de portal functie, het type aanbieding en de organisatie van de publicatie. De resultaten kunnen worden gerangschikt op kolom of gefilterd op een opgegeven teken reeks.  

Op deze pagina wordt ook een knop **gebruiker toevoegen** weer gegeven voor het toevoegen van gebruikers aan uw publicatie organisatie.  Zie [gebruikers beheren op het Cloud Partner-Portal](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-users.md)voor meer informatie.
