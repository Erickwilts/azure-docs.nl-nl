# <a name="get-started-with-azure-security-center-for-iot"></a>Aan de slag met Azure Security Center for IoT

In dit artikel wordt uitleg gegeven over de verschillende onderdelen van de Azure Security Center for IoT-service en wordt uitgelegd hoe u met de service aan de slag met behulp van twee mogelijke implementatieopties.

## <a name="deployment-options"></a>Implementatieopties

Kies het servicescenario dat het beste voldoet aan de vereisten van uw IoT-apparaat en omgeving.

### <a name="built-in-deployment"></a>Ingebouwde implementatie

Met de naadloze, ingebouwde implementatieoptie kan Azure Security Center for IoT snel worden ge√Øntegreerd in uw IoT-hub en beveiligingsanalyses bieden van de IoT-hubconfiguratie, apparaatidentiteit en -beheer en communicatiepatronen op hub-apparaten.

Start een [ingebouwde implementatie](iot-hub-integration.md) met IoT Hub-bewaking en aanbevelingen.
    <br>

### <a name="enhanced-deployment"></a>Verbeterde implementatie

Voor verbeterde beveiligingsmogelijkheden biedt het implementeren van Azure Security Center voor IoT-agents naast het inschakelen van IoT Hub-beveiliging op basis van een agent het verzamelen, analyseren en detecteren van belangrijke beveiligingsgegevens van uw IoT-apparaten en uitgebreide mogelijkheden voor het beheer van beveiligingshoudingen.

Start een [verbeterde implementatie](security-agents.md) met een uitgebreide oplossing voor uitgebreide bedreigingsbescherming en beveiligingshoudingsbeheer op basis van een agent.
