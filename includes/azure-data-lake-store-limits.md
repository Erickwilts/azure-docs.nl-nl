Azure Data Lake Storage Gen1 is een bedrijfsbrede hyper-scale repository voor big data analytische workloads. U Data Lake Storage Gen1 gebruiken om gegevens van elke grootte, elk type en opnamesnelheid op één plaats vast te leggen voor operationele en verkennende analyses. Er is geen limiet aan de hoeveelheid gegevens die u opslaan in een Data Lake Storage Gen1-account.

| **Resource** | **Limiet** | **Opmerkingen** |
| --- | --- | --- |
| Maximaal aantal Data Lake Storage Gen1-accounts, per abonnement, per regio |10 | Als u een verhoging voor deze limiet wilt aanvragen, neemt u contact op met de ondersteuning. |
| Maximum aantal toegangsACL's, per bestand of map |32 | Dit is een harde limiet. Gebruik groepen om toegang te beheren met minder vermeldingen. |
| Maximum aantal standaard ACL's, per bestand of map |32 | Dit is een harde limiet. Gebruik groepen om toegang te beheren met minder vermeldingen. |
