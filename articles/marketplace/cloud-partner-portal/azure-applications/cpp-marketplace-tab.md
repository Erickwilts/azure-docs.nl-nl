# <a name="azure-application-marketplace-tab"></a>Tabblad Azure Application Marketplace

De [Cloud Partner-Portal](https://cloudpartner.azure.com/) biedt geen ondersteuning meer voor het maken en beheren van Azure-toepassingen. Deze functionaliteit is in plaats daarvan verplaatst naar de portal van micro soft [Partner Center](https://partner.microsoft.com/). Zie [een Azure-toepassings aanbieding maken](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-new-azure-apps-offer)voor meer informatie.
