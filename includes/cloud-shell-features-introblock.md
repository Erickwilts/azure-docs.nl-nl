Azure Cloud Shell is een op de browser gebaseerde shell-ervaring voor het beheren en ontwikkelen van Azure-resources.

Cloud Shell biedt een browsertoegankelijke, vooraf geconfigureerde shell-ervaring voor het beheren van Azure-resources zonder de overhead van het installeren, versien en onderhouden van een machine zelf.

Cloud Shell-provisionsmachines per aanvraag en als gevolg daarvan blijven de machinestatus niet voor alle sessies bestaan. Aangezien Cloud Shell is gebouwd voor interactieve sessies, eindigen shells automatisch na 20 minuten shell inactiviteit.
