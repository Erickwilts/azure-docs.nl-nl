# <a name="what-is-azure-netapp-files"></a>Wat is Azure NetApp Files?

De service Azure NetApp Files is een bestandsopslagservice op enterprise-niveau met hoge prestaties die naar gebruik wordt gefactureerd. Azure NetApp Files biedt ondersteuning voor elk type werkbelasting en is standaard maximaal beschikbaar. U kunt service- en prestatieniveaus selecteren en instellen van momentopnamen via de service. 

## <a name="learn-how-to-use-azure-netapp-files"></a>Meer informatie over het gebruik van Azure NetApp bestanden

De documentatie van Azure NetApp Files biedt instructies voor het maken en beheren van volumes met behulp van Azure NetApp Files. 

> [!IMPORTANT] 
> Alle taken die worden beschreven in deze documentatie moeten u toegang tot de service Azure NetApp bestanden hebben gekregen. Zie voor het aanvragen van toegang tot de service, [een wachtlijst indienen voor toegang tot de service](azure-netapp-files-register.md#waitlist).
