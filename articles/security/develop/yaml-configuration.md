# <a name="yaml-configuration-options-to-customize-the-build-tasks"></a>YAML-configuratieopties om de buildtaken aan te passen

In dit artikel worden alle YAML-configuratieopties weergegeven die beschikbaar zijn in elk van de buildtaken. Het artikel begint met de taken voor beveiligingscodeanalysetools. Het eindigt met de nabewerkingstaken.
