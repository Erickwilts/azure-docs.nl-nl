# <a name="unity-showcase-application"></a>Unity-presentatietoepassing

De Unity-presentatietoepassing is een op Unity gebaseerde voorbeeldtoepassing voor het demonstreren van Azure Remote Rendering-functies. De toepassing kan worden uitgevoerd in de Unity-editor en op Hololens 2.

![Presentatie-app](./media/showcase-app.png "Presentatie-app")

## <a name="source-repository"></a>Bronopslagplaats

De broncode voor deze toepassing vindt u, met de instructies voor bouwen/ontwikkelen, in de [opslagplaats met ARR-samples](https://github.com/Azure/azure-remote-rendering/tree/master/Unity/Showcase).
