# <a name="what-tools-are-included-on-the-azure-data-science-virtual-machine"></a>Welke hulpprogramma's zijn opgenomen op de Azure Data Science Virtual Machine?

De Data Science Virtual Machine is de eenvoudigste manier om gegevens te verkennen en machine learning in de cloud te doen. De Data Science Virtual Machines zijn vooraf geconfigureerd met het volledige besturingssysteem, beveiligingspatches, stuurprogramma's en populaire data science en development software. U kiezen voor de hardwareomgeving, variërend van goedkopere CPU-centrische machines tot zeer krachtige machines met meerdere GPU's, NVMe-opslag en grote hoeveelheden geheugen. Voor machines met GPU's zijn alle stuurprogramma's geïnstalleerd, zijn alle machine learning-frameworks versie-matched voor GPU-compatibiliteit en is versnelling ingeschakeld in alle toepassingssoftware die GPU's ondersteunt.

De Data Science Virtual Machine wordt geleverd met de meest nuttige data-science tools vooraf geïnstalleerd:

## <a name="how-to-choosecondaenvironment"></a>Hoe conda-omgeving te kiezen  

> [!NOTE]
> Er zijn brekende veranderingen bij het overschakelen naar de nieuwe versie van Data Science Virtual Machine van de oude versie. 

**Overschakelen van Windows Server 2016 naar Windows Server 2019** </br> 

* De conda-omgeving is ingesteld op python37_default. We ondersteunen python2 niet. Als u gebruik maken van tensorflow of pytorch in uw python conda omgeving gebruik py37_tensorflow of py37_pytorch respectievelijk. Als u alleen AzureML-gebruik azureml_py36_automl </br>

* Als u tensorflow of pytorch gebruikt in uw AzureML-conda-omgeving, gebruikt u respectievelijk azureml_py36_tensorflow of azureml_py36_pytorch. </br>

**Overschakelen van Ubuntu 16.04 naar Ubuntu 18.04** </br>

* Als u gebruik maakt van tensorflow of pytorch in uw azureml_py36 conda omgeving gebruik azureml_py36_tensorflow of azureml_py36_pytorch respectievelijk. </br>

* Als u gebruik maakt van tensorflow of pytorch in uw py37 conda omgeving gebruik py37_tensorflow of py37_pytorch respectievelijk. </br>

* Gebruik anders python37_default of gebruik u alleen azureml_py36 azureml_py36_automl </br>
