# <a name="introduction-to-the-deep-learning-virtual-machine"></a>Inleiding tot de uitgebreide virtuele machine van Learning

## <a name="why-deep-learning-virtual-machine"></a>Waarom is de virtuele machine dieper leren? 

Diep gaande algoritmen/diepe Neural-netwerken worden steeds vaker gebruikt voor een van de populaire methoden die in veel problemen met machine learning worden toegepast. Ze zijn vooral geschikt voor cognition-taken zoals afbeeldings-, tekst-, audio-en video-inzichten die vaak de microcognitieve niveaus in sommige specifieke domeinen benaderen met geavanceerde diepe Neural-netwerk architecturen en toegang tot grote set gegevens om modellen te trainen. Voor diepe training is een grote hoeveelheid reken kracht nodig om modellen met deze grote gegevens sets te trainen. Met de Cloud en beschik baarheid van grafische verwerkings eenheden (Gpu's) is het mogelijk om geavanceerde diepe Neural architecturen te bouwen en ze te trainen op een grote gegevensset op een krachtige computer infrastructuur in de Cloud.  De [Data Science virtual machine](overview.md) beschikt over een uitgebreide set hulpprogram ma's en voor beelden voor gegevens voorbereiding, machine learning en diep gaande lessen. Maar een van de uitdagingen die de gebruikers bevallen, is om de hulpprogram ma's en voor beelden te ontdekken voor specifieke scenario's, zoals het snel leren en het inrichten van op GPU gebaseerde VM-exemplaren. In deze DLVM (diep gaande training) worden deze uitdagingen verholpen. 

## <a name="what-is-deep-learning-virtual-machine"></a>Wat is een diep gaande virtuele machine? 
De diep gaande virtuele machine is een speciaal geconfigureerde variant van de [Data Science virtual machine](overview.md) (DSVM), zodat deze eenvoudiger te gebruiken is voor het gebruik van op GPU gebaseerde VM-instanties voor het trainen van diepe leer modellen. Het wordt ondersteund op Windows 2016 en de Ubuntu-Data Science Virtual Machine.  Het deelt dezelfde kern installatie kopieën van de virtuele machine (en dus alle uitgebreide hulpprogram ma's) als de DSVM, maar is zo geconfigureerd dat het eenvoudiger wordt om dieper te leren. We bieden ook end-to-end-voor beelden voor het begrip van installatie kopieën en teksten die breed toepasbaar zijn voor veel Real-Life AI-scenario's. De diep gaande virtuele machine probeert ook om de uitgebreide set hulpprogram ma's en voor beelden op de DSVM eenvoudiger te vinden door een catalogus van de hulpprogram ma's en voor beelden op de virtuele machine te halen. In termen van het hulp programma biedt de virtuele machine van het diepe leer proces diverse populaire diep gaande frameworks, hulpprogram ma's voor het verkrijgen en vooraf verwerken van afbeeldingen, tekst gegevens. Voor een uitgebreide lijst met hulpprogram ma's kunt u verwijzen naar de [overzichts pagina van data Science virtual machine](overview.md#whats-included-in-the-data-science-vm).
