### <a name="azure-active-directory-authentication"></a>Azure Active Directory-verificatie

Azure Spatial Anchors maakt het ook mogelijk toepassingen te verifiëren met gebruiker Azure AD (Active Directory) tokens. U bijvoorbeeld Azure AD-tokens gebruiken om te integreren met Azure Spatial Anchors. Als een Onderneming gebruikers in Azure AD onderhoudt, u een Azure AD-token voor gebruikers leveren in de Azure Spatial Anchors SDK. Hiermee u zich rechtstreeks verifiëren naar de Azure Spatial Anchors-service voor een account dat deel uitmaakt van dezelfde Azure AD-tenant.
