# <a name="jenkins-plugins-for-azure"></a>Jenkins-invoeg toepassingen voor Azure

De volgende Jenkins-invoeg toepassingen ondersteunen verschillende functies voor gebruik met Azure.
  
| Jenkins-invoeg toepassing | Beschrijving                                   |
|------------------------------------------------------------------------------|
| [Azure App Service-invoeg toepassing](https://plugins.jenkins.io/azure-app-service)     | Jenkins-invoeg toepassing voor het implementeren van een Azure App Service (momenteel ondersteunt alleen web-app). | 
| [Azure AD-invoeg toepassing](https://plugins.jenkins.io/azure-ad)                       | Jenkins-invoeg toepassing die verificatie & autorisatie via Azure Active Directory ondersteunt. | 
| [Azure artefact beheer-invoeg toepassing](https://plugins.jenkins.io/azure-artifact-manager) | De Azure artefact beheer-invoeg toepassing is een artefact beheerder waarmee u uw artefacten kunt opslaan in Azure Blob Storage. De invoeg toepassing voor Azure-artefact beheer werkt op transparante wijze samen voor Jenkins en uw taken. Dit is vergelijkbaar met het standaard artefact beheer. | 
| [Azure Container Agents-invoegtoepassing](https://plugins.jenkins.io/azure-container-agents) | Met de invoeg toepassing Azure container agents kunt u een container als agent uitvoeren in Jenkins | 
| [Invoeg toepassing Azure Container Registry taak](https://plugins.jenkins.io/azure-container-registry-tasks)       | Jenkins-invoeg toepassing voor het verzenden van een docker-build-aanvraag naar [Azure container Registry](/azure/container-registry/container-registry-tasks-overview). |
| [Azure Container Service-invoegtoepassing](https://plugins.jenkins.io/azure-acs)       | Jenkins-invoeg toepassing voor het implementeren van configuraties voor Azure Container Service (AKS). | 
| [Azure Credential-invoegtoepassing](https://plugins.jenkins.io/azure-credentials)      | Jenkins-invoeg toepassing voor het beheren van Azure-referenties. | 
| [Azure Function-invoegtoepassing](https://plugins.jenkins.io/azure-function)           | Jenkins-invoeg toepassing voor het implementeren van een Azure-functie. | 
| [Azure IoT Edge-invoeg toepassing](https://plugins.jenkins.io/azure-iot-edge)           | Azure IoT Edge-invoeg toepassing maakt het eenvoudig om een CI/CD-pijp lijn in te stellen voor het ontwikkelen van een IoT Edge oplossing op Jenkins. | 
| [Invoeg toepassing voor Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Jenkins-invoeg toepassing voor Linux Azure Service Fabric-projecten. |
| [Azure Storage-invoegtoepassing](https://plugins.jenkins.io/windows-azure-storage)     | -invoeg toepassing voor het uploaden van build-artefacten naar of downloaden van BUILD-afhankelijkheden van Microsoft Azure Blob-opslag. | 
| [Invoeg toepassing voor Azure VM-agents](https://plugins.jenkins.io/azure-vm-agents)         | Jenkins-invoeg toepassing voor het maken van Jenkins-agents in azure virtual machines (via Azure Resource Manager-sjabloon). |
