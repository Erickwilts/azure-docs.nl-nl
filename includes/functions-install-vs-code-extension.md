## <a name="install-the-azure-functions-extension"></a>De azure-functie-extensie installeren

U de azure-functie-extensie gebruiken om functies te maken en te testen en deze te implementeren in Azure.

1. Open in Visual Studio Code **extensies** en zoek naar **azure-functies**of [selecteer deze koppeling in Visual Studio Code](vscode:extension/ms-azuretools.vscode-azurefunctions).

1. Selecteer **Installeren** om de extensie voor Visual Studio Code te installeren:

    ![De extensie voor Azure Functions installeren](./media/functions-install-vs-code-extension/vscode-install-extension.png)

1. Selecteer na de installatie het Azure-pictogram op de activiteitsbalk. Het gebied Azure Functions moet nu in de zijbalk worden weergegeven.

    ![Gebied Azure Functions in de zijbalk](./media/functions-install-vs-code-extension/azure-functions-window-vscode.png)
