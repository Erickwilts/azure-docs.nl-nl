# <a name="unity-showcase-application"></a>Unity-presentatietoepassing

De toepassing unit show case is een voor beeld-app op basis van een eenheid voor het demonstreren van de functies voor externe rendering van Azure. Het kan worden uitgevoerd in de Unity-editor en op Hololens 2.

![App laten opvallen](./media/showcase-app.png "App laten opvallen")

## <a name="source-repository"></a>Bron opslagplaats

De bron code voor deze toepassing, naast instructies voor Build/Deployment, vindt u in de [opslag plaats voor beelden](https://github.com/Azure/azure-remote-rendering/tree/master/Unity/AzureRemoteRenderingShowcase).
