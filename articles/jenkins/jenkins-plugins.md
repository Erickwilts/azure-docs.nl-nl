# <a name="jenkins-plugins-for-azure"></a>Jenkins-plug-ins voor Azure

De volgende Jenkins-plug-ins ondersteunen verschillende functies voor gebruik met Azure.
  
| Jenkins-plug-in | Beschrijving                                   |
|------------------------------------------------------------------------------|
| [Azure App Service-plug-in](https://plugins.jenkins.io/azure-app-service)     | Jenkins-plug-in om een Azure App-service te implementeren (ondersteunt momenteel alleen Web App). | 
| [Azure AD-invoegtoepassing](https://plugins.jenkins.io/azure-ad)                       | Jenkins-plug-in die verificatie ondersteunt & autorisatie via Azure Active Directory. | 
| [Azure Artifact Manager-plug-in](https://plugins.jenkins.io/azure-artifact-manager) | Azure Artifact Manager-plug-in is een Artefact-manager waarmee u uw artefacten opslaan in Azure Blob Storage. Azure Artifact Manager-plug-in werkt transparant voor Jenkins en uw taken, het is als de standaard Artefact Manager. | 
| [Azure Container Agents-invoegtoepassing](https://plugins.jenkins.io/azure-container-agents) | Azure Container Agents-plug-in kan u helpen om een container als agent in Jenkins uit te voeren | 
| [Plug-in Azure Container Registry Task](https://plugins.jenkins.io/azure-container-registry-tasks)       | Jenkins-plug-in om een dockerbuildaanvraag naar [Azure Container Registry](/azure/container-registry/container-registry-tasks-overview)te verzenden. |
| [Azure Container Service-invoegtoepassing](https://plugins.jenkins.io/azure-acs)       | Jenkins-plug-in om configuraties te implementeren voor Azure Container Service (AKS). | 
| [Azure Credential-invoegtoepassing](https://plugins.jenkins.io/azure-credentials)      | Jenkins-plug-in om Azure-referenties te beheren. | 
| [Azure Function-invoegtoepassing](https://plugins.jenkins.io/azure-function)           | Jenkins-plug-in om een Azure-functie te implementeren. | 
| [Azure IoT Edge-plug-in](https://plugins.jenkins.io/azure-iot-edge)           | Azure IoT Edge-plug-in maakt het eenvoudig om een CI/CD-pijplijn in te stellen voor het ontwikkelen van een IoT Edge-oplossing op Jenkins. | 
| [Azure Service Fabric-invoegtoepassing](https://plugins.jenkins.io/service-fabric)     | Jenkins-plug-in voor Linux Azure Service Fabric-projecten. |
| [Azure Storage-invoegtoepassing](https://plugins.jenkins.io/windows-azure-storage)     | plug-in voor het uploaden van buildartefacten naar of het downloaden van buildafhankelijkheden van Microsoft Azure Blob-opslag. | 
| [Azure VM-agents-plug-in](https://plugins.jenkins.io/azure-vm-agents)         | Jenkins-plug-in om Jenkins-agents te maken in virtuele Azure-machines (via azure resource manager-sjabloon). | 
| [Plug-in azure voor het schalen van virtuele machines](https://plugins.jenkins.io/azure-vmss)           | Jenkins-plug-in om VM-afbeeldingen te implementeren in Azure-set's voor virtuele machines. |
