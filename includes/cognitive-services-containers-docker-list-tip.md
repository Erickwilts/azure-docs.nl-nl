> [!TIP]
> U de opdracht [dockerafbeeldingen](https://docs.docker.com/engine/reference/commandline/images/) gebruiken om uw gedownloade containerafbeeldingen weer te geven. In de volgende opdracht worden bijvoorbeeld de id, opslagplaats en tag van elke gedownloade containerafbeelding weergegeven, die is opgemaakt als een tabel:
>
>  ```
>  docker images --format "table {{.ID}}\t{{.Repository}}\t{{.Tag}}"
>
>  IMAGE ID         REPOSITORY                TAG
>  <image-id>       <repository-path/name>    <tag-name>
>  ```
