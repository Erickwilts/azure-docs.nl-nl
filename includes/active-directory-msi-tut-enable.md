Het inschakelen van een door het systeem toegewezen beheerde identiteit is een ervaring met één klik. U kunt deze inschakelen tijdens het maken van een virtuele machine of in de eigenschappen van een bestaande virtuele machine.

![Nieuw opslagaccount maken](./media/active-directory-msi-tut-enable/identity.png)


**Een door het systeem toegewezen beheerde identiteit inschakelen op een nieuwe virtuele machine:** 

1. [Aanmelden bij Azure Portal](https://portal.azure.com)

2. [Een virtuele machine maken met door het systeem toegewezen identiteit ingeschakeld](/azure/active-directory/managed-identities-azure-resources/qs-configure-portal-windows-vm#system-assigned-managed-identity)
