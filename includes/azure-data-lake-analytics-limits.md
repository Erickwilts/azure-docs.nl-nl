Azure Data Lake Analytics maakt de complexe taak van het beheren van gedistribueerde infrastructuur en complexe code eenvoudig. Het bevat dynamisch resources en u deze gebruiken om analyses uit te doen op exabytes aan gegevens. Wanneer de taak is voltooid, worden de resources automatisch afgebroken. U betaalt alleen voor de gebruikte verwerkingskracht. Naarmate u de grootte van de opgeslagen gegevens of de gebruikte hoeveelheid rekenkracht verhoogt of verkleint, hoeft u de code niet te herschrijven. Neem contact op met de ondersteuning om de standaardlimieten voor uw abonnement te verhogen.

| **Resource** | **Limiet** | **Opmerkingen** |
| --- | --- | --- |
| Maximum aantal gelijktijdige taken |20 | |
| Maximaal aantal analyse-eenheden (AU's) per account |250 | Gebruik een combinatie van maximaal 250 AU's voor 20 vacatures. Neem contact op met Microsoft Support om deze limiet te verhogen. |
| Maximale scriptgrootte voor taakindiening | 3 MB | |
| Maximaal aantal Data Lake Analytics-accounts per regio per abonnement | 5 | Neem contact op met Microsoft Support om deze limiet te verhogen. |
