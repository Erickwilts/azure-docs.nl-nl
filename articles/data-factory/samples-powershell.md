# <a name="azure-powershell-samples-for-azure-data-factory"></a>Azure PowerShell-voorbeelden voor Azure Data Factory

De volgende tabel bevat koppelingen naar Azure PowerShell-voorbeeldscripts voor Azure Data Factory.

| |  |
|---|---|
|**Gegevens kopiëren**||
|[Blobs kopiëren vanuit een map naar een andere map in een Azure Blob-opslag](scripts/copy-azure-blob-powershell.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Dit PowerShell-script worden blobs uit een map in Azure Blob-opslag gekopieerd naar een andere map in dezelfde blobopslag. |
|[Gegevens van on-premises SQL Server naar Azure Blob Storage kopiëren](scripts/hybrid-copy-powershell.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Dit PowerShell-script kopieert gegevens uit een on-premises SQL Server-database naar een Azure blob-opslag. |
|[Bulksgewijs kopiëren](scripts/bulk-copy-powershell.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Met dit PowerShell-voorbeeldscript kopieert gegevens uit meerdere tabellen in een Azure SQL-database naar een Azure SQL datawarehouse. |
|[Incrementele kopie](scripts/incremental-copy-powershell.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Met dit PowerShell-voorbeeldscript worden alleen nieuwe of bijgewerkte records van een brongegevensarchief naar een sink-gegevensarchief geladen na de eerste volledige kopie van gegevens uit de bron voor de sink. |
|**Gegevens transformeren**||
|[Gegevens transformeren met behulp van een Spark-cluster](scripts/transform-data-spark-powershell.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Dit PowerShell-script transformeert gegevens met het uitvoeren van een programma op een Spark-cluster. |
|**Lift- and -shift SSIS-pakketten naar Azure**||
|[Azure-SSIS integratieruntime maken](scripts/deploy-azure-ssis-integration-runtime-powershell.md?toc=%2fpowershell%2fmodule%2ftoc.json)| In dit PowerShell-script bepalingen een Azure-SSIS integratieruntime die SQL Server Integration Services (SSIS)-pakketten in Azure uitvoert. |
