# <a name="azure-operational-security"></a>Operationele Azure-beveiliging
## <a name="abstract"></a>Samenvatting
Microsoft Azure operationele beveiliging verwijst naar de services, besturings elementen en functies die beschikbaar zijn voor gebruikers voor het beveiligen van hun gegevens, toepassingen en andere assets in Azure. Azure Operational Security is gebaseerd op een Framework met de kennis die is opgedaan via verschillende mogelijkheden die uniek zijn voor micro soft, waaronder micro soft Security Development Lifecycle (SDL), het micro soft Security Response Center-programma, en dieper inzicht in de Cyber beveiliging Threat landschap. Dit technisch document geeft een overzicht van de manier waarop u operationele beveiliging kunt benaderen met behulp van Azure. Het omvat verschillende Azure-Services, waaronder:
* Azure Monitor-logboeken
* Azure Backup
* Azure Security Center
* Azure Monitor
* Azure Network Watcher
* Azure Storage Analytics
* Azure Active Directory
