# <a name="scenario-unable-to-log-into-azure-hdinsight-cluster"></a>Scenario: kan niet aanmelden bij het Azure HDInsight-cluster

In dit artikel worden de stappen beschreven voor het oplossen van problemen en mogelijke oplossingen voor problemen bij het werken met Azure HDInsight-clusters.

## <a name="issue"></a>Probleem

Kan niet aanmelden bij het Azure HDInsight-cluster.

## <a name="cause"></a>Oorzaak

Redenen kunnen variÃ«ren. Als u zich aanmeldt bij de cluster-of app-Dash boards, gebruikt u de aanmeldings gegevens voor het cluster of HTTP. Gebruik bij een verbinding op afstand uw referenties voor Secure Shell (SSH) of Extern bureaublad.

## <a name="resolution"></a>Oplossing

Probeer een of meer van de volgende stappen om veelvoorkomende problemen op te lossen.

* Open het clusterdashboard op een nieuw browsertabblad in de privacymodus.

* Als u uw SSH-referenties niet kunt intrekken, kunt u [de referenties opnieuw instellen in de Ambari-gebruikers interface](../hdinsight-administer-use-portal-linux.md#change-passwords).
