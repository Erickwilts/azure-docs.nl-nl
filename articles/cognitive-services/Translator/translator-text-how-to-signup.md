# <a name="how-to-sign-up-for-the-translator-text-api"></a>Aanmelden voor de Translator Text API

## <a name="sign-in-to-the-azure-portal"></a>Aanmelden bij Azure Portal

- Heb je nog geen account? U kunt een [gratis account](https://azure.microsoft.com/free/) maken als u kosteloos wilt experimenteren.
- U heeft een account, u bent abonnee? [Aanmelden](https://ms.portal.azure.com/)

## <a name="create-a-subscription-to-the-translator-text-api"></a>Een abonnement maken op de Translator Text API

Nadat u zich bij de portal hebt aangemeld, u als volgt een abonnement op de Translator Text API maken:

1. Selecteer **+ Een resource maken**.
1. Typ in het zoekvak **Zoeken in De Marktplaats** de tekst van **vertalers** en selecteer deze in de resultaten.
1. Selecteer **Maken** om details voor het abonnement te definiëren.
1. Selecteer **in** de lijst Met prijzen de prijscategorie die het beste bij uw behoeften past.
    1. Elk abonnement heeft een gratis laag. De gratis laag heeft dezelfde functies en functionaliteiten als de betaalde abonnementen en verloopt niet.
    1. Je maar één gratis abonnement voor je account hebben.
1. Selecteer **Maken** om het maken van het abonnement te voltooien.

## <a name="authentication-key"></a>Verificatiesleutel

Wanneer u zich aanmeldt voor Translator Text, krijgt u een persoonlijke toegangssleutel die uniek is voor uw abonnement. Deze sleutel is vereist voor elke aanroep naar de Translator Text API.

1. Haal uw verificatiesleutel op door eerst het juiste abonnement te selecteren.
1. Selecteer **Sleutels** in de sectie **Resourcebeheer** van de gegevens van uw abonnement.
1. Kopieer een van de sleutels die voor uw abonnement worden vermeld.

## <a name="learn-test-and-get-support"></a>Leren, testen en ondersteuning krijgen

- [Codevoorbeelden op GitHub](https://github.com/MicrosoftTranslator)

Microsoft Translator laat uw eerste paar aanvragen over het algemeen passeren voordat de status van het abonnementsaccount is geverifieerd. Als de eerste paar API-aanvragen voor Microsoft Translator slagen, mislukken de aanroepen, wordt het probleem probleem aangegeven. Log de API-reactie zodat u de reden zien.
