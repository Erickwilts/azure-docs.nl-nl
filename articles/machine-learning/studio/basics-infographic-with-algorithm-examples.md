# <a name="downloadable-infographic-machine-learning-basics-with-algorithm-examples"></a>Downloadbare Infographic: basisbeginselen van Machine Learning met algoritmevoorbeelden

[!INCLUDE [Notebook deprecation notice](../../../includes/aml-studio-notebook-notice.md)]

Download dit eenvoudig te begrijpen infographic-overzicht van de basisprincipes van machine learning om meer te weten te komen over populaire algoritmen die worden gebruikt om veelgestelde vragen over machine learning te beantwoorden. Algoritmevoorbeelden helpen de beginner van machine learning te begrijpen welke algoritmen ze moeten gebruiken en waarvoor ze worden gebruikt.

## <a name="popular-algorithms-in-machine-learning-studio-classic"></a>Populaire algoritmen in Machine Learning Studio (klassiek)

[Azure Machine Learning Studio (klassiek)](https://studio.azureml.net/) wordt geleverd met een grote bibliotheek met algoritmen voor voorspellende analyses. Deze infographic identificeert vier populaire families van algoritmen - regressie, anomaliedetectie, clustering en classificatie - en biedt koppelingen naar werkvoorbeelden in de [Azure AI Gallery](https://gallery.azure.ai/). De Galerij bevat voorbeeldexperimenten en zelfstudies die aantonen hoe deze algoritmen kunnen worden toegepast in veel real-world oplossingen.
