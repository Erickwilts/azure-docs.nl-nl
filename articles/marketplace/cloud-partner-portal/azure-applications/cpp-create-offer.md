# <a name="create-an-azure-application-offer"></a>Een Azure-toepassingsaanbieding maken

In dit artikel wordt beschreven hoe u een Azure-toepassingsaanbiedingsvermelding voor de Azure Marketplace maken en publiceren. Elk aanbod wordt weergegeven als een eigen entiteit in Azure Marketplace en bestaat uit de volgende groeperingen van activa en ondersteunende services: bied details, SKU's, marketingartefacten en ondersteuningsinformatie aan.

## <a name="new-offer-form"></a>Nieuw aanbiedingsformulier

Meld u aan bij de [Cloud Partner Portal](https://cloudpartner.azure.com/)en selecteer + Nieuwe **aanbieding** op de linker menubalk. Selecteer **azure-toepassingen** in het menu Nieuwe aanbieding om het formulier **Nieuwe aanbieding** weer te geven en het proces van het definiëren van elementen voor een nieuwe aanbieding te starten.

![Nieuw aanbiedingsformulier](./media/azure-new-managedapp-offer.png)
