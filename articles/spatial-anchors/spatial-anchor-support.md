# <a name="azure-spatial-anchors-support-options"></a>Ondersteuningsopties voor Azure Spatial Anchors

## <a name="open-a-tech-support-ticket"></a>Open een ticket voor technische ondersteuning

Ga als het gaat om een ticket voor technische ondersteuning te openen in de Azure Portal for Azure Spatial Anchors:

1. Als de [Azure-portal](https://azure.microsoft.com/account/) is geopend, selecteert u het Help-pictogram in de bovenste menubalk en selecteert u de knop **Help + ondersteuning.**

![Help en ondersteuning voor Azure-portalen](./media/spatial-anchor-support.png)

1. Als de help + ondersteuningspagina is geopend, selecteert u **+ Nieuw ondersteuningsverzoek**.

![Nieuwe ondersteuningsaanvraag azure portal](./media/spatial-anchor-support2.png)

1. Bij het invullen van de velden voor ondersteuningstickets:

- Type probleem: Technisch
- Service: Ruimtelijke ankers

![Ticketvelden azure portal-ondersteuning](./media/spatial-anchor-support3.png)
