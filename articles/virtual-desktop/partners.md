## <a name="citrix"></a>Citrix

![Citrix-logo](./media/partners/citrix.png)

Citrix is een goedgekeurde provider die bedrijven gecentraliseerd hybride beheer van virtuele apps en desktops workloads in Azure biedt, zij aan zij met on-premises implementaties. Citrix Workspace met de Virtual Apps and Desktops-service stelt gebruikers in staat om vanaf elk apparaat toegang te krijgen tot apps en desktops, waarbij gebruik wordt gemaakt van het geavanceerde Citrix HDX-protocol om overal een hd-ervaring te bieden.

Citrix breidt de waarde van Windows Virtual Desktop uit met robuuste bedrijfstools om de gebruikersdichtheid en -prestaties te verbeteren, workloads op aanvraag te bieden en het beheer van afbeeldingen en toepassingen te vereenvoudigen. IT kan de kosten optimaliseren met intelligente schaalgereedschappen, terwijl het een ongelooflijke gebruikerservaring biedt die is getest tegen de zwaarste toepassingen in verschillende sectoren. Daarnaast is Citrix Managed Desktops een Windows Virtual Desktop-enabled desktops-as-a-service programma dat een eenvoudige, cloudgebaseerde beheeroplossing biedt voor het leveren van virtuele apps en desktops op elk apparaat.

## <a name="vmware"></a>VMware

![VMware-logo](./media/partners/vmware.png)

VMware Horizon Cloud op Microsoft Azure biedt organisaties de mogelijkheid om hun eigen exemplaar van Microsoft Azure te verbinden met het eenvoudige, intuïtieve Horizon Cloud-besturingsvlak, waardoor een veilige, uitgebreide, cloudgehoste oplossing wordt gemaakt voor het leveren van gevirtualiseerde Windows-toepassingen en desktops. Deze functionaliteit zal eind 2019 beschikbaar zijn in Tech Preview.
 
Als goedgekeurde provider worden de voordelen van Windows Virtual Desktop uitgebreid naar VMware Horizon Cloud op Microsoft Azure-klanten, waaronder Windows 10 Enterprise voor meerdere sessies, Windows 7 met gratis uitgebreide beveiligingsupdates voor maximaal drie jaar en FSLogix.
