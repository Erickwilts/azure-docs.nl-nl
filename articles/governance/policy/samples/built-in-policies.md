# <a name="azure-policy-built-in-policy-definitions"></a>Ingebouwde beleids definities Azure Policy

Deze pagina bevat een index van Azure Policy ingebouwde beleids definities.

De naam van elke ingebouwde koppeling naar de beleids definitie in Azure Portal. Gebruik de koppeling in de kolom **bron** om de bron weer te geven op het [Azure Policy github opslag plaats](https://github.com/Azure/azure-policy). De ingebouwde invoeg toepassingen worden gegroepeerd op basis van de eigenschap **Category** in **meta data**. Als u naar een specifieke **categorie**wilt gaan, gebruikt u het menu aan de rechter kant van de pagina. Als dat niet het geval is, gebruikt u <kbd>Ctrl</kbd>-<kbd>F</kbd> om de zoek functie van uw browser te gebruiken.
