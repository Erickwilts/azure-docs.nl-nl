# <a name="scenario-hdp-upgrade-causes-no-data-in-apache-phoenix-views-in-azure-hdinsight"></a>Scenario: HDP-upgrade zorgt ervoor dat er geen gegevens worden gegevens in Apache Phoenix-weergaven in Azure HDInsight

In dit artikel worden stappen voor het oplossen van problemen en mogelijke oplossingen voor problemen beschreven bij interactie met Azure HDInsight-clusters.

## <a name="issue"></a>Probleem

Een Apache Phoenix-weergave bevat geen datum na een upgrade van HDP 2.4 naar HDP 2.5.

## <a name="cause"></a>Oorzaak

De indextabel voor weergaven (alle indexen voor weergave worden opgeslagen in één fysieke Apache HBase-tabel) wordt tijdens de upgrade afgekapt

## <a name="resolution"></a>Oplossing

Drop en recreëer alle weergave-indexen na de upgrade.
