# <a name="create-a-new-container-offer-with-the-cloud-partner-portal"></a>Maak een nieuwe container aanbieding met de Cloud Partner-portal

In dit artikel wordt beschreven hoe u een vermelding voor een container-aanbod maakt en publiceert voor Azure Marketplace. Elk aanbod wordt als eigen entiteit weer gegeven in azure Marketplace en is gekoppeld aan een of meer Sku's.  Een container aanbod bestaat uit de volgende groeperingen van assets en ondersteunende services:

|  **Activa groep**   |  **Beschrijving**  |
|  ---------------   |  ---------------  |
|    Voorraad            |  De kleinste inzet bare eenheid van een aanbieding. Voor een enkele aanbieding (product klasse) kunnen meerdere Sku's aan de aanbieding zijn gekoppeld. U kunt Sku's gebruiken om onderscheid te maken tussen ondersteunde functies en facturerings modellen. |
|  Marketplace       | Bevat marketing-, juridische-en Lead beheer activa en-specificaties.  <ul><li> Marketing assets bevatten naam, beschrijving en logo's van de aanbieding</li> <li> Juridische activa omvatten een privacybeleid, gebruiks voorwaarden en andere juridische documentatie</li>  <li> Met het beleid voor het beheer van leads kunt u opgeven hoe leads van de eind gebruikers portal van Azure Marketplace moeten worden afgehandeld.</li> </ul> |
| Ondersteuning            | Bevat ondersteunings contactpersoon en beleids gegevens |


## <a name="new-offer-form"></a>Formulier voor nieuwe aanbieding 

Meld u aan bij de [Cloud Partner-Portal](https://cloudpartner.azure.com/)en selecteer **+ nieuwe aanbieding** op de menu balk aan de linkerkant. Selecteer in het menu nieuwe aanbieding de optie **containers** om het **nieuwe aanbod** formulier weer te geven en het proces voor het definiëren van assets voor een nieuwe container aanbieding te starten.

![Selecteer de optie containers voor de nieuwe aanbieding](./media/azure-container-offer.png)
