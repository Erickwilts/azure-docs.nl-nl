# <a name="azure-data-factory-whitepapers"></a>Azure Data Factory technische documenten 

Technische documenten kunt u Azure Data Factory op een dieper niveau verkennen. Dit artikel bevat een lijst met beschikbare technische documenten voor Azure Data Factory.

| **Technisch document** | **Beschrijving** |
| --- | --- |
|[Azure Data Factory: integratie van gegevens in de Cloud](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | In dit artikel wordt beschreven hoe Azure Data Factory kunt u een moderne datawarehouse opzetten, schakelt u geavanceerde analyses aan intelligente SaaS-toepassingen werken en uw SQL Server-integratieservices-pakketten naar Azure brengen.|
|[Migratie van de gegevens van on-premises relationele Data Warehouse naar Azure met behulp van Azure Data Factory](https://azure.microsoft.com/mediahandler/files/resourcefiles/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/Data_migration_from_on-prem_RDW_to_ADLS_using_ADF.pdf) | In dit artikel wordt de complexiteit van de migratie van tientallen TB gegevens van bestaande on-premises relationele datawarehouse (bijvoorbeeld Netezza, Oracle, Teradata, SQL server) met behulp van Azure Data Factory naar Azure (bijvoorbeeld Blob Storage of Azure Data Lake Storage). De uitdagingen en aanbevolen procedures worden weergegeven rond de flexibiliteit, prestaties, schaalbaarheid, beheer en beveiliging voor de opname op weg big data naar Azure met Azure Data Factory. |
|[Azure Data Factory: SSIS in de Cloud](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| In dit artikel gaat over waarom u wilt migreren van uw bestaande SSIS-werkbelastingen naar Azure Data Factory- en algemene overwegingen bij de adres- en problemen. We vervolgens helpt u bij de technische details van het maken van een Azure-SSIS IR en ziet u hoe u uploaden, uitvoeren en bewaken van uw pakketten via de Azure Data Factory met behulp van de hulpprogramma's die u waarschijnlijk bekend met, zoals SQL Server Management Studio (SSMS bent). |
|[Azure data Factory-Parameters doorgeven](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| In dit artikel leert u hoe u parameters doorgeven tussen een pijplijn en activiteit ook tussen de activiteiten. |
|[Azure data Factory-DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| In dit technische document bevat informatie over enkele van de aanbevolen procedures behandeld voor continue integratie en implementatie van Azure Data Factory. |
