# <a name="cheat-sheet-for-an-automated-data-pipeline-for-azure-machine-learning-predictions"></a>Overzichtskaart voor een geautomatiseerde pijplijn voor voorspellingen van Azure Machine Learning
Met het **spiekbedrogblad voor geautomatiseerde gegevenspijplijn van Microsoft Azure Machine Learning** u navigeren door de technologie die u gebruiken om uw gegevens naar uw Machine Learning-webservice te brengen, waar deze kunnen worden gescoord door uw voorspellende analysemodel.

Afhankelijk van of uw gegevens on-premises, in de cloud of real-time streaming zijn, zijn er verschillende mechanismen beschikbaar om de gegevens naar het eindpunt van uw webservice te verplaatsen om te scoren.
Dit spiekbriefje leidt u door de beslissingen die u moet maken, en het biedt links naar artikelen die u kunnen helpen bij het ontwikkelen van uw oplossing.
