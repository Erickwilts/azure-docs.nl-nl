# <a name="manage-your-azure-maps-account"></a>Uw Azure Maps-account beheren

U uw Azure Maps-account beheren via de Azure-portal. Nadat u een account hebt, u de API's implementeren in uw website of mobiele applicatie.

Als u geen Azure-abonnement hebt, maakt u een [gratis account](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) voordat u begint.

## <a name="create-a-new-account"></a>Een nieuw account maken

1. Meld u aan bij [Azure Portal](https://portal.azure.com).

2. Selecteer **Een resource maken** in de linkerbovenhoek van Azure Portal.

3. Zoeken naar en selecteer **Kaarten**. Selecteer vervolgens **Maken**.

4. Voer de gegevens in voor uw nieuwe account.

[![Azure Maps-accountgegevens invoeren in de Azure-portal](./media/how-to-manage-account-keys/new-account-portal.png)](./media/how-to-manage-account-keys/new-account-portal.png#lightbox)

## <a name="delete-an-account"></a>Een account verwijderen

U een account verwijderen uit de Azure-portal. Navigeer naar de pagina accountoverzicht en selecteer **Verwijderen**.

[![Uw Azure Maps-account verwijderen in de Azure-portal](./media/how-to-manage-account-keys/account-delete-portal.png)](./media/how-to-manage-account-keys/account-delete-portal.png#lightbox)

U ziet dan een bevestigingspagina. U het verwijderen van uw account bevestigen door de naam ervan te typen.
