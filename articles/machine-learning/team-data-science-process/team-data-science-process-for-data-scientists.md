# <a name="team-data-science-process-for-data-scientists"></a>Team Data Science Process voor data scientists

In dit artikel vindt u richtlijnen voor een reeks doelstellingen die doorgaans worden gebruikt om uitgebreide data science-oplossingen met Azure-technologieën te implementeren. U wordt begeleid door:

- inzicht in een analytics-workload
- het Team Data Science-proces gebruiken
- Azure Machine Learning gebruiken
- de fundamenten van gegevensoverdracht en -opslag
- het verstrekken van gegevensbrondocumentatie
- tools gebruiken voor analyseverwerking

Deze trainingsmaterialen zijn gerelateerd aan het Team Data Science Process (TDSP) en Microsoft en open-source software en toolkits, die nuttig zijn voor het bedenken, uitvoeren en leveren van data science-oplossingen.
