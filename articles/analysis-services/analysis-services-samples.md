# <a name="azure-analysis-services-samples"></a>Azure Analysis Services-voorbeelden

Gebruik de volgende voorbeeldbronnen om informatie te krijgen over analyseservices voor uw omgeving en deze te testen.

## <a name="code-samples"></a>Codevoorbeelden

De [Analysis Services](https://github.com/Microsoft/Analysis-Services) repository op GitHub bevat open source code samples en community projecten. 

## <a name="tabular-model-project-and-database-samples"></a>Tabelmodelproject- en databasevoorbeelden

[Adventure Works for Analysis Services](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks-analysis-services) op GitHub is het meest gebruikte voorbeeldmodelproject. U een VS-project of een voltooide voorbeeldtabelmodeldatabase downloaden.

## <a name="sample-database-on-azure"></a>Voorbeelddatabase op Azure

Azure Synapse Analytics (SQL Data Warehouse) biedt een voorbeeld van AdventureWorksDW-database die kan worden opgenomen in een ingerichte bron. Zie [Snelstart: Een AZURE Synapse Analytics SQL-groep maken en opvragen](/azure/sql-data-warehouse/create-data-warehouse-portal)voor meer informatie.

## <a name="sample-databases-on-github"></a>Voorbeelddatabases op GitHub

Deze voorbeelddatabases op GitHub kunnen worden gebruikt voor het maken en testen van uw eigen modellen. 

[Voorbeelddatabases van Adventure Works](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks2014)

[Wide World Importers monsterdatabases](https://github.com/Microsoft/sql-server-samples/releases/tag/wide-world-importers-v1.0)

## <a name="adventure-works-internet-sales-sample-model-in-azure-portal"></a>Voorbeeldmodel voor internetverkoop van Adventure Works in Azure-portal

Als u een Analysis Services-server in azure-portal hebt, u snel en eenvoudig een voorbeeldmodel maken. Klik op de overzichtspagina van uw server in azure-portal op **Nieuw model**en selecteer vervolgens in **Voorbeeldgegevens**kiezen **.**
