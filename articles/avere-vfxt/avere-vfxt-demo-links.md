# <a name="avere-vfxt-demo-projects"></a>Avere vFXT demo projecten

Voorbeeldtutorials worden aangeboden op [GitHub.](https://github.com/Azure/Avere) Deze kleine projecten tonen belangrijke functies en use cases voor Avere vFXT voor Azure.

## <a name="video-rendering"></a>Videorendering

* [Rendering met Azure Batch en Avere vFXT](https://github.com/Azure/Avere/blob/master/docs/maya_azure_batch_avere_vfxt_demo.md) - Een 60 minuten durend project dat laat zien hoe je Autodesk Maya gebruiken met Azure Batch en een Avere vFXT-cluster om een animatiefilm te genereren

* [Waarom de Avere vFXT gebruiken voor rendering?](https://github.com/Azure/Avere/blob/master/docs/why_avere_for_rendering.md) - Een demonstratie die rendertijden van netwerkopslag vergelijkt met en zonder een Avere vFXT-cluster

## <a name="high-performance-computing"></a>High-performance computing

* [Aanbevolen procedures voor het verbeteren van de opstarttijd van Azure Virtual Machine (VM)](https://github.com/Azure/Avere/blob/master/docs/azure_vm_provision_best_practices.md) - Een reproduceerbare test die Avere vFXT voor Azure gebruikt om de opstarttijden te verbeteren wanneer duizenden compute clients snel worden opgestart

## <a name="vfxt-performance"></a>vFXT-prestaties

* [Meet vFXT-prestaties met vdbench](https://github.com/Azure/Avere/blob/master/docs/vdbench.md) - Een eenvoudige testopstelling om kleine en middelgrote workloads te genereren om de vFXT-geheugen- en schijfsubsystemen te testen

## <a name="client-setup"></a>Clientinstelling

* [Windows 10-workstation voor Avere vFXT](https://github.com/Azure/Avere/blob/master/docs/windows_10_avere_vfxt_mounted_workstation.md) - Laat zien hoe je een Windows-workstation instelt en monteer het op een Avere vFXT-cluster
