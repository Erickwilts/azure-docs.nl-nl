### <a name="azure-active-directory-authentication"></a>Azure Active Directory-verificatie

Met Azure Spatial Anchors kunnen toepassingen zich ook verifiëren met gebruikerstokens van Azure AD (Active Directory). U kunt bijvoorbeeld Azure AD-tokens gebruiken voor integratie met Azure Spatial Anchors. Als een onderneming gebruikers onderhoudt in Azure AD, kunt u een Azure AD-gebruikerstoken verstrekken in de Azure Spatial Anchors SDK. Dit stelt u in staat om rechtstreeks te verifiëren bij de Azure Spatial Anchors-service voor een account dat deel uitmaakt van dezelfde Azure AD-tenant.
