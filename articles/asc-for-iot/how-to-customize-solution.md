# <a name="customize-your-azure-security-center-for-iot-solution"></a>Uw Azure Security Center voor IoT-oplossing aanpassen

In deze hand leiding vindt u informatie over het aanpassen van verschillende instellingen in Azure Security Center voor IoT.

> [!div class="checklist"]
> * Aanbevelingen voor oplossingen configureren
> * Instellingen wijzigen

## <a name="configure-solution-recommendations"></a>Aanbevelingen voor oplossingen configureren

Ga als volgt te werk om uw Azure Security Center voor de aanbevelingen van IoT-oplossingen te configureren:

1. Open uw **IOT hub** in azure Portal.
1. Selecteer en open **instellingen** onder **beveiliging** in het linkermenu.
1. Selecteer bij **instellingen**de optie **aanbevolen configuraties**.
1. Schakel de aanbevelingen voor oplossingen uit/in die relevant zijn voor uw organisatie en werk stromen.

## <a name="change-settings"></a>Instellingen wijzigen

Uw Azure Security Center voor IoT-instelling beheren:

Ga als volgt te werk om uw Azure Security Center voor IoT-instellingen te configureren:

1. Open uw **IOT hub** in azure Portal.
1. Selecteer in het menu links onder **beveiliging**de optie **instellingen**.
1. Onder * * gegevens verzameling * * * * selecteert u de oplossings instelling die u wilt wijzigen.
1. Klik altijd op **Opslaan** boven aan een instellings scherm om uw instellings wijzigingen op te slaan.
