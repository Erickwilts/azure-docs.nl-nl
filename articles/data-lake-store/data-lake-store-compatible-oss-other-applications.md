# <a name="open-source-big-data-applications-that-work-with-azure-data-lake-storage-gen1"></a>Open Source Big Data-toepassingen die met Azure Data Lake Storage Gen1 werken

[!INCLUDE [data-lake-storage-gen1-rename-note.md](../../includes/data-lake-storage-gen1-rename-note.md)]

In dit artikel geeft een lijst van de open-source big data-toepassingen die met Azure Data Lake Storage Gen1 werken. Voor de toepassingen in de onderstaande tabel zijn alleen de versies die beschikbaar zijn met de vermelde distributiepunten worden ondersteund. Zie voor informatie over welke versies van deze toepassingen beschikbaar met HDInsight zijn, [versiebeheer van HDInsight-onderdeel](../hdinsight/hdinsight-component-versioning.md).

| [Apache Storm](https://storm.apache.org/) |HDInsight 3.2, 3.4, 3.5 en 3.6 |
| [Apache Hive](https://hive.apache.org/) |HDInsight 3.2, 3.4, 3.5 en 3.6 |
