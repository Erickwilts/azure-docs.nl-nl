# <a name="azure-maps-community---open-source-projects"></a>Azure Maps-community - Opensourceprojecten

Deze open-source, community-gedreven initiatieven worden gemaakt en onderhouden door het Azure Maps-team. Ze maken geen deel uit van het standaardproduct- of serviceaanbod.

In de volgende lijsten enkele van de meest populaire Azure Maps open-source projecten en voorbeelden.

| Projectnaam | Beschrijving |
|-|-|
| [Voorbeelden van Azure Maps-code](https://github.com/Azure-Samples/AzureMapsCodeSamples) | Een verzameling codevoorbeelden voor het gebruik van Azure Maps in webgebaseerde apps. |
| [Voorbeelden van Azure Maps Gov-cloudcode](https://github.com/Azure-Samples/AzureMapsCodeSamples) | Een verzameling codevoorbeelden voor het gebruik van Azure Maps via Azure Government Cloud. |
| [Azure Maps & Azure Active Directory-voorbeelden](https://github.com/Azure-Samples/Azure-Maps-AzureAD-Samples) | Een verzameling voorbeelden die laten zien hoe u Azure Active Directory gebruiken met Azure Maps. | 
| [Ruimtelijke IoT-analyse implementeren met behulp van Azure Maps](https://github.com/Azure-Samples/iothub-to-azure-maps-geofencing) | Het bijhouden en vastleggen van relevante gebeurtenissen die zich voordoen in ruimte en tijd is een veelvoorkomend IoT-scenario. |
| [Voorbeelden van Azure Maps Jupyter-notitieblokken](https://github.com/Azure-Samples/Azure-Maps-Jupyter-Notebook) | Een verzameling python-voorbeelden met de Azure Maps REST-services. |
| [Bot Framework - Point of Interest vaardigheid](https://github.com/microsoft/botframework-solutions/tree/488093ac2fddf16096171f6a926315aa45e199e7/skills/csharp/pointofinterestskill) | De Point of Interest Skill biedt POI-gerelateerde mogelijkheden aan een virtuele assistent met Azure Maps met Azure Bot Service en Bot Framework. |

Een langere lijst met community-aanmaakprogramma's is [hier](https://github.com/microsoft/Maps/blob/master/AzureMaps.md) beschikbaar

## <a name="supportability-of-open-source-projects"></a>Draagbaarheid van open-source projecten

De volgende instructies zijn van toepassing op alle Azure Maps die opensourceprojecten en -voorbeelden hebben gemaakt en onderhouden:

- Azure Maps open source-projecten en -voorbeelden worden gemaakt door Microsoft en de community.
- Azure Maps open-source projecten en voorbeelden worden onderhouden door Microsoft en de gemeenschap.
- Azure Maps opensourceprojecten en -voorbeelden maken gebruik van ondersteunde en aanbevolen technieken.
- Azure Maps open-source projecten en voorbeelden zijn een community-initiatief - mensen die werken aan het initiatief ten behoeve van anderen, en hebben hun normale baan ook.
- Azure Maps open-source projecten en voorbeelden zijn geen product, en daarom wordt het niet ondersteund door Premier Support of andere officiële ondersteuningskanalen.
- Azure Maps open-source projecten en voorbeelden worden ondersteund op dezelfde manier als andere open-source projecten gedaan door Microsoft met ondersteuning van de gemeenschap door de gemeenschap.
