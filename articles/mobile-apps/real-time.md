# <a name="build-real-time-applications-with-azure-signalr"></a>Real-time toepassingen bouwen met Azure-Signa lering

Met [Azure signalr](https://azure.microsoft.com/services/signalr-service/) kunt u eenvoudig real-time functionaliteit aan toepassingen toevoegen. Met deze real-time functionaliteit kan de service inhouds updates naar verbonden mobiele toepassingen pushen. Als gevolg hiervan worden apps bijgewerkt zonder dat de server moet worden gecontroleerd of nieuwe HTTP-aanvragen voor updates moeten worden verzonden.

## <a name="azure-signalr-features"></a>Functies van Azure signalering
- **Uitgebreide ondersteuning voor clients** : het werkt met een breed scala aan clients, zoals web-en mobiele browsers, desktop toepassingen, server processen, IOT-apparaten en spel consoles.
- Met de Signa lering-service kunnen **meerdere instanties samen werken** om te schalen naar miljoenen mobiele toepassings verbindingen. Het ondersteunt ook meerdere globale regio's voor sharding, hoge Beschik baarheid of herstel na nood gevallen.
- **Bieden uitgebreide api's voor verschillende bericht patronen** voor het verzenden van berichten naar een bepaalde verbinding, alle verbindingen of een subset van verbindingen die tot een specifieke gebruiker behoren of die in een wille keurige groep zijn geplaatst.
