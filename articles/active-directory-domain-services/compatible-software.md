# <a name="third-party-software-compatible-with-azure-ad-domain-services"></a>Software van derden die compatibel is met Azure AD Domain Services

## <a name="one-identity-authentication-services"></a>Eén identiteitsverificatieservices

![Eén identiteit](./media/compatible-software-icons/OneIdentity.png)

Met One Identity Authentication Services kunnen Unix-, Linux- en Mac OS X-systemen de toegang, verificatie en autorisatie van de bestaande Ad-infrastructuur (Active Directory) van een organisatie gebruiken. Authentication Services ondersteunt nu Azure Active Directory Domain Services waarmee niet-Windows-bronnen hetzelfde platform van de volgende generatie kunnen gebruiken dat uw bestaande SaaS-oplossingen al gebruiken.

[Eén identiteitsverificatieservices](https://www.quest.com/products/authentication-services/)

[KB-artikel - Eén identiteitsverificatieservices gebruiken met Azure AD Domain Services](https://support.quest.com/authentication-services/kb/208427)


## <a name="red-hat-enterprise-linux"></a>Red Hat Enterprise Linux

![Red Hat](./media/compatible-software-icons/RedHat.png)

Red Hat Enterprise Linux is 's werelds toonaangevende enterprise Linux-platform gebouwd om te voldoen aan de behoeften van de moderne onderneming van vandaag. Red Hat Enterprise Linux is de voorkeurskeuze voor virtuele machineworkloads (Enterprise Linux) op Microsoft Azure. Red Hat Enterprise Linux is een open, betrouwbaar en veilig platform dat is ontworpen voor klanten die implementatieflexibiliteit willen voor hun bedrijfskritieke workloads - van het datacenter tot de Azure-cloud - ondersteund door nauw geïntegreerde ondersteuning op bedrijfsniveau van Red Hat en Microsoft.
