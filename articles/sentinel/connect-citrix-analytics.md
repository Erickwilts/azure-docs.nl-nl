# <a name="connect-citrix-analytics-security"></a>Verbinding maken met Citrix Analytics (Security) 

Citrix Analytics (Security) integratie met Azure Sentinel helpt u bij het exporteren van gegevens die zijn geanalyseerd voor risicovolle gebeurtenissen van Citrix Analytics (Security) naar Azure Sentinel. U aangepaste dashboards maken, gegevens uit andere bronnen analyseren, samen met die van Citrix Analytics (Beveiliging) en aangepaste workflows maken met Behulp van Logic Apps om beveiligingsgebeurtenissen te bewaken en te beperken. 



## <a name="prerequisites"></a>Vereisten

- Lees- en schrijfmachtigingen voor de Azure Sentinel-werkruimte.

- Recht op Citrix Security Analytics in Citrix Cloud.


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>Azure Sentinel verbinden met Citrix Analytics (Beveiliging)

1. Navigeer in de **Azure-portal** > naar Azure Sentinel**Data-connectors** en selecteer vervolgens de **Citrix Analytics -connector (Security).**

2. Selecteer **Connectorpagina openen**.

3. Volg de instructies op de **citrix Analytics (Security)** pagina.
