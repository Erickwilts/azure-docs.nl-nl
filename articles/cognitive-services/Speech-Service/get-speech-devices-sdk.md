# <a name="get-the-cognitive-services-speech-devices-sdk"></a>De Cognitive Services spraak apparaten SDK ophalen

De spraak-apparaten-SDK is een pretuned bibliotheek is ontworpen voor gebruik met speciaal ontwikkelde development kits en diverse configuraties van de microfoon matrix.
