# <a name="azure-policy-built-in-definitions-for-azure-stream-analytics"></a>Ingebouwde Azure Policy-definities voor Azure Stream Analytics

Zie [Ingebouwde Azure Policy-definities](../governance/policy/samples/built-in-policies.md) voor aanvullende ingebouwde modules voor Azure Policy voor andere services.
