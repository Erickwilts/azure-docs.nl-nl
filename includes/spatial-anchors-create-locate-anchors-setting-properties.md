## <a name="set-properties"></a>Eigenschappen instellen

U kunt ervoor kiezen om sommige eigenschappen toe te voegen bij het opslaan van de ruimte in de Cloud. Net als het type van het object dat wordt opgeslagen of basis eigenschappen, zoals of het moet worden ingeschakeld voor interactie. Dit kan handig zijn bij detectie: u kunt het object direct voor de gebruiker weer geven, bijvoorbeeld een afbeeldings frame met lege inhoud. Vervolgens krijgt een andere down load op de achtergrond aanvullende status Details, bijvoorbeeld de afbeelding die in het kader moet worden weer gegeven.
