### <a name="open-the-mfa-page"></a><a name="mfa"></a>De MFA-pagina openen

1. Meld u aan bij Azure Portal.
2. Navigeer naar **Azure Active Directory -> Alle gebruikers**.
3. Selecteer **Meervoudige verificatie** om de pagina met meervoudige verificatie te openen.

   ![Aanmelden](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa1.jpg)

### <a name="select-users"></a><a name="users"></a>Gebruikers selecteren

1. Selecteer op de pagina **multi-factor authenticatie** de gebruiker(s) voor wie u MFA wilt inschakelen.
2. Selecteer **Inschakelen**.

   ![Selecteer](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa2.jpg)
