# <a name="saas-fulfillment-apis-version-1-deprecated"></a>SaaS-fulfillment-Api's versie 1 (afgeschaft)

> [!WARNING]
> Deze eerste versie van de SaaS-fulfillment-API is afgeschaft. gebruik in plaats daarvan [SaaS fulfillment API v2](./pc-saas-fulfillment-api-v2.md).  Deze eerste versie van de API wordt momenteel alleen onderhouden voor bestaande uitgevers. 

De documentatie voor deze versie van de API is niet meer beschikbaar. Als u van mening bent dat u toegang nodig hebt, kunt u contact opnemen met saas-api-v1@microsoft.com .

Voor alle nieuwe ontwikkelingen gaat u naar [v2-versie van de API](./pc-saas-fulfillment-api-v2.md) , zelfs als u v1 al gebruikt. Deze versie wordt in de toekomst volledig afgeschaft.
