# <a name="delete-a-group-using-azure-active-directory"></a>Een groep verwijderen met Azure Active Directory
U een Azure Active Directory-groep (Azure AD) verwijderen om verschillende redenen, maar meestal komt dit doordat u:

- Stel het **groepstype** ten onrechte in op de verkeerde optie.

- Gemaakt van de verkeerde of een dubbele groep per ongeluk. 

- Ik heb de groep niet meer nodig.

## <a name="to-delete-a-group"></a>Een groep verwijderen
1. Meld u aan bij de [Azure Portal](https://portal.azure.com) met het account van een globale administrator voor de map.

2. Selecteer **Azure Active Directory**en selecteer **Groepen**.

3. Zoek in de pagina **Groepen - Alle groepen** naar de groep die u wilt verwijderen. Voor deze stappen gebruiken we **mdm-beleid - Oost**.

    ![Pagina Groepen-Alle groepen, groepsnaam gemarkeerd](media/active-directory-groups-delete-group/group-all-groups-screen.png)

4. Selecteer op de pagina **MDM-beleid - Oostoverzicht** en selecteer **Verwijderen**.

    De groep wordt verwijderd uit uw Azure Active Directory-tenant.

    ![MDM-beleid - Pagina Overzicht Oost, optie verwijderen gemarkeerd](media/active-directory-groups-delete-group/group-overview-blade.png)

## <a name="next-steps"></a>Volgende stappen

- Als u een groep per ongeluk verwijdert, u deze opnieuw maken. Zie [Een basisgroep maken en leden toevoegen voor](active-directory-groups-create-azure-portal.md)meer informatie.

- Als u een Office 365-groep per ongeluk verwijdert, u deze mogelijk herstellen. Zie [Een verwijderde Office 365-groep herstellen](../users-groups-roles/groups-restore-deleted.md)voor meer informatie .
