# <a name="overview-of-the-pci-dss-v321-blueprint-sample"></a>Overzicht van het blauwdrukvoorbeeld voor PCI-DSS v3.2.1

Het blauwdrukvoorbeeld voor PCI-DSS v3.2.1 is een set beleidsregels die bijdragen aan de naleving voor PCI-DSS v3.2.1. Deze blauwdruk helpt klanten cloud-omgevingen met PCI-DSS-workloads te beheren.
Met de PCI-DSS-blauwdruk is een kernset beleidsregels te implementeren voor elke Azure-architectuur die aan deze accreditatie moet voldoen.

## <a name="control-mapping"></a>Toewijzing van beheeropties

De sectie over toewijzing van beheeropties biedt meer informatie over de beleidsregels die in dit initiatief zijn opgenomen en hoe dit beleid kan bijdragen aan verschillende beheeropties die zijn gedefinieerd in PCI-DSS v3.2.1. Wanneer resources worden toegewezen aan een architectuur, worden deze op niet-naleving van toegewezen beleidsregels geÃ«valueerd door Azure Policy.

Na het toewijzen van deze blauwdruk kunt u het nalevingsniveau van uw Azure-omgevingen weergeven in het Azure-dashboard voor beleidsnaleving.
