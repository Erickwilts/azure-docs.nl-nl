# <a name="get-started-with-azure-security-center-for-iot"></a>Aan de slag met Azure Security Center for IoT

Dit artikel bevat een uitleg van de verschillende onderdelen van de Azure Security Center voor de IoT-service en legt uit hoe u aan de slag kunt met de service met twee mogelijke implementatie opties.

Bent u op zoek naar aan de slag met de beveiligings module voor Azure RTO'S? Ga naar de sectie [Snelstartgids voor beveiligings module voor Azure rto's](quickstart-azure-rtos-security-module.md) . 

## <a name="deployment-options"></a>Implementatieopties

Kies het service scenario dat het beste voldoet aan uw IoT-apparaat en omgevings vereisten.

### <a name="built-in-deployment"></a>Ingebouwde implementatie

Met de naadloze, ingebouwde implementatie optie Azure Security Center voor IoT snel kan worden ge√Øntegreerd in uw IoT Hub en kan de configuratie van de IoT-hub, de identiteit en het beheer van apparaten en de communicatie patronen voor Hub-apparaten worden verbeterd.

Start een [ingebouwde implementatie](iot-hub-integration.md) met IOT hub bewaking en aanbevelingen.
    <br>

### <a name="enhanced-deployment"></a>Verbeterde implementatie

Voor verbeterde beveiligings mogelijkheden, het implementeren van Azure Security Center voor IoT-agents en het inschakelen van IoT Hub beveiliging biedt gebeurtenissen verzameling op basis van een agent, analyse en detectie van bedreigingen van belang rijke beveiligings gegevens van uw IoT-apparaten, evenals uitgebreide mogelijkheden voor het beheer van beveiligings postuur.

Start een [verbeterde implementatie](security-agents.md) met een uitgebreide oplossing voor beveiliging tegen bedreigingen en postuur beheer van een agent.
