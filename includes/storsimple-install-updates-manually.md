#### <a name="to-install-regular-updates-via-the-azure-classic-portal"></a>Regel matige updates installeren via de klassieke Azure-Portal
1. Selecteer op de pagina **apparaten** het apparaat waarop u updates wilt installeren.
2. Navigeer naar **apparaten**  >  **onderhoud** en schuif omlaag naar **software-updates**.
3. Als u op updates wilt controleren, klikt u onder aan de pagina op **updates controleren** .
4. Er wordt een bericht weer gegeven als er software-updates beschikbaar zijn. Klik op **updates installeren** om het apparaat bij te werken.
