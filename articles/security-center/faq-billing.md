# <a name="billing-questions"></a>Vragen over facturering

## <a name="how-does-billing-work-for-azure-security-center"></a>Hoe werkt facturering voor Azure Security Center?
Security Center wordt aangeboden in twee prijscategorieÃ«n:

- De **gratis laag** biedt inzicht in de beveiligingsstatus van uw Azure-bronnen, basisbeveiligingsbeleid, beveiligingsaanbevelingen en integratie met beveiligingsproducten en -services van partners.

- De **standaardlaag** voegt mogelijkheden voor bedreigingsbescherming toe, waaronder beveiligingswaarschuwingen, bedreigingsinformatie, gedragsanalyse, anomaliedetectie en meldingsrapporten voor het toeschrijvingsgebied van bedreigingen. U een gratis proefversie van de standaardlaag starten. Als u wilt upgraden, selecteert u [Prijsniveau](https://docs.microsoft.com/azure/security-center/security-center-pricing) in het beveiligingsbeleid. Zie de [pagina met prijzen](https://azure.microsoft.com/pricing/details/security-center/) voor meer informatie.

## <a name="how-can-i-track-who-in-my-organization-performed-pricing-tier-changes-in-azure-security-center"></a>Hoe kan ik bijhouden wie in mijn organisatie wijzigingen in de prijscategorie heeft uitgevoerd in Azure Security Center
Azure-abonnementen kunnen meerdere beheerders hebben met machtigingen om de prijscategorie te wijzigen. Als u wilt weten welke gebruiker een prijswijzigen heeft uitgevoerd, gebruikt u het Azure-activiteitenlogboek. Voor meer informatie, zie [hier](https://techcommunity.microsoft.com/t5/Security-Identity/Tracking-Changes-in-the-Pricing-Tier-for-Azure-Security-Center/td-p/390832).
