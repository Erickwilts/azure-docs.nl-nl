# <a name="using-the-azure-importexport-tool"></a>Het hulpprogramma voor Importeren/exporteren van Azure gebruiken

Het Azure Import/Export Tool (WAImportExport.exe) wordt gebruikt voor het maken en beheren van taken voor de Azure Import/Export-service, zodat u grote hoeveelheden gegevens overbrengen naar of uit Azure Blob Storage.

Deze documentatie is voor de meest recente versie van het Azure Import/Export Tool. Zie [Azure Import/Export Tool v1](storage-import-export-tool-how-to-v1.md)gebruiken voor informatie over het gebruik van het klassieke implementatiemodel.

In de volgende artikelen ziet u hoe u:  

- Installeer en stel het Azure Import/Export Tool in.
- Bereid uw harde schijven voor op een taak waarbij u gegevens van uw schijven importeert naar Azure Blob Storage.
