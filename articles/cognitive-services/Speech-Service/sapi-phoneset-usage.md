# <a name="sapi-phone-set-usage"></a>Gebruik van SAPI Phone set

Micro soft definieert SAPI Phone set voor zeven talen (en-US, fr-FR, de-DE, es-ES, ja-JP, zh-CN, zh-TW). De SAPI Phone-set is relatief eenvoudig te gebruiken en is vooral voor eigen sprekers te onthouden. Bijvoorbeeld ja-JP gebruiken Kana als telefoon, zh-CN gebruik PinYin als telefoonset. Maar soms is het niet krachtig genoeg, bijvoorbeeld door de SAPI-telefoonset fr-FR geen Franse samen werking te vertegenwoordigen.
