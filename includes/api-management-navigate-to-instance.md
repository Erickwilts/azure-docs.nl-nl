## <a name="go-to-your-api-management-instance"></a>Ga naar uw API Management-exemplaar

1. Zoek in de Azure-portal naar **API Management-services** en selecteer dit.

   ![API Management-services selecteren](media/api-management-navigate-to-instance/view-apim1.png)

1. Selecteer in het venster **API Management** uw API Management-exemplaar.

   ![Uw API Management-exemplaar selecteren](media/api-management-navigate-to-instance/view-apim2.png)
