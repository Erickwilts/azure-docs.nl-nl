# <a name="service-model-xml-schema-simple-types"></a>Eenvoudige typen XML-schema voor service model

## <a name="fabricuri-simpletype"></a>FabricUri Simple type
Een URI die wordt gebruikt als een stabiele id van services door Microsoft Azure Service Fabric naamgevings systeem. 

### <a name="xml-source"></a>XML-bron
```xml
<xs:simpleType xmlns:xs="https://www.w3.org/2001/XMLSchema" xmlns="http://schemas.microsoft.com/2011/01/fabric" name="FabricUri">
    <xs:annotation>
      <xs:documentation>A URI that is used as a stable identifier of services by Microsoft Azure Service Fabric Naming system. </xs:documentation>
    </xs:annotation>
    <xs:restriction base="xs:anyURI"/>
  </xs:simpleType>
  
```
