# <a name="azure-policy-built-in-policy-definitions"></a>Ingebouwde beleidsdefinities voor Azure-beleid

Deze pagina is een index van ingebouwde beleidsdefinities van Azure Policy.

De naam van elke ingebouwde koppeling naar de beleidsdefinitie in Azure-portal. Gebruik de koppeling in de kolom **Bron** om de bron op de [Azure Policy GitHub repo](https://github.com/Azure/azure-policy)weer te geven. De ingebouwde ins worden gegroepeerd op de **categorieeigenschap** in **metagegevens.** Als u naar een specifieke **categorie**wilt gaan, gebruikt u het menu aan de rechterkant van de pagina. Gebruik <kbd>anders Ctrl</kbd>-<kbd>F</kbd> om de zoekfunctie van uw browser te gebruiken.
