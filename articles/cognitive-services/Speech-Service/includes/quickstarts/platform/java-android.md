In deze gids ontdekt u hoe u de [Speech SDK](~/articles/cognitive-services/speech-service/speech-sdk.md) kunt installeren voor Java op Android. De installatie is gebaseerd op het pakket Speech SDK Maven en Android Studio 3.3.

> [!NOTE]
> Zie [Speech Devices SDK](~/articles/cognitive-services/speech-service/speech-devices-sdk.md) voor de Speech Devices-SDK en het Roobo-apparaat.

[!INCLUDE [License Notice](~/includes/cognitive-services-speech-service-license-notice.md)]

## <a name="supported-platforms"></a>Ondersteunde platforms

De Speech SDK is compatibel met Android-apparaten met 32/64-bits ARM- en Intel x86/x64-compatibele processors.
