Als u de cmdlets wilt uitvoeren, u Azure Cloud Shell gebruiken, een gratis interactieve shell. In deze shell zijn algemene Azure-hulpprogramma's vooraf geïnstalleerd en geconfigureerd voor gebruik met uw account. Klik op **Kopiëren** om de code te kopiëren, plak deze in Cloud Shell en druk vervolgens op Enter om de code uit te voeren. U kunt Cloud Shell op verschillende manieren starten:


|  |   |
|-----------------------------------------------|---|
| Open Cloud Shell in uw browser. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Klik op de knop **Cloud Shell** in het menu rechtsboven in de portal. | [![Cloud Shell in de portal](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Als u Azure CloudShell niet wilt gebruiken, u PowerShell lokaal installeren. Als u ervoor kiest PowerShell lokaal te installeren en te gebruiken, moet u de nieuwste versie van de PowerShell-cmdlets van Resource Manager installeren. PowerShell-cmdlets worden regelmatig bijgewerkt en u moet uw PowerShell-cmdlets meestal bijwerken om de nieuwste functiefunctionaliteit te krijgen, bij gebreke daarvan, u problemen ondervinden.

Gebruik de cmdlet 'Get-Module -ListAvailable Az' om de versie van PowerShell te vinden die u lokaal uitvoert. Zie De [Azure PowerShell-module installeren](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps)als u deze wilt bijwerken. Zie [Azure PowerShell installeren en configureren](https://docs.microsoft.com/powershell/azure/azurerm/overview) voor meer informatie.

Als u PowerShell op macOS gebruikt, volgt u de stappen om [PowerShell Core op macOS te installeren.](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6)
