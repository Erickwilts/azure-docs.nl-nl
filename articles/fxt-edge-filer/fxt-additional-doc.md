# <a name="additional-documentation"></a>Aanvullende documentatie 

Sommige bronnen buiten deze docs.microsoft.com website kunnen u helpen uw microsoft Azure FXT Edge Filer-hybride cache te begrijpen en ermee te werken.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Avere vFXT voor Azure-cachedocumentatie

[Avere vFXT voor Azure](https://docs.microsoft.com/azure/avere-vfxt/) is een vergelijkbaar clustercacheproduct dat virtuele knooppunten in Microsoft Azure gebruikt. Het is gemaakt met de Azure-portal, maar maakt gebruik van dezelfde configuratiesoftware voor het configuratiescherm als de Azure FXT Edge Filer-hardwarecache. 

## <a name="legacy-hardware-documentation"></a>Verouderde hardwaredocumentatie 

Deze technologie werd eerder gebruikt in producten van Avere Systems. Gebruikersdocumentatie voor die oudere producten is beschikbaar op de [website van Avere Legacy Documentation.](https://azure.github.io/Avere/) 

In het bijzonder kunnen deze documenten nuttige details bevatten:

* [Clusterconfiguratiehandleiding](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) - Bevat overzichtsinformatie over algemene clusterinstellingen en beheertaken en bevat gedetailleerde uitleg over de pagina's met clusterinstellingen 

* [Dashboardgids](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) - Legt de functies van het **tabblad Dashboard** van het configuratiescherm uit

* [FXT-clustercreatiegids](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) - Handleiding voor het maken van clusteren van eerdere producten
