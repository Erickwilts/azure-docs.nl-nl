# <a name="azure-cli-samples-for-azure-cosmos-db-mongodb-api"></a>Azure CLI-voor beelden voor Azure Cosmos DB MongoDB-API

De volgende tabel bevat koppelingen naar Azure CLI-voorbeeld scripts voor Azure Cosmos DB MongoDB-API. Referentie pagina's voor alle Azure Cosmos DB CLI-opdrachten zijn beschikbaar in de [Naslag informatie voor Azure cli](/cli/azure/cosmosdb). Alle Azure Cosmos DB CLI-voorbeeld scripts vindt u in de [Azure Cosmos DB cli github-opslag plaats](https://github.com/Azure-Samples/azure-cli-samples/tree/master/cosmosdb).

> [!NOTE]
> Op dit moment kunt u alleen 3,2-versie maken (dat wil zeggen, accounts die gebruikmaken `*.documents.azure.com`van het eind punt in de indeling) van de API van Azure Cosmos DB voor MongoDb-accounts met behulp van Power shell, CLI en Resource Manager-sjablonen. Als u 3,6-versie van accounts wilt maken, gebruikt u Azure Portal in plaats daarvan.
