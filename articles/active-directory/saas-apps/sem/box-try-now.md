# <a name="box-and-azure-active-directory"></a>Vak en Azure Active Directory

Versnel de acceptatie van box in uw organisatie en verg root de acceptatie van gebruikers met onze vooraf ontwikkelde integratie. Gebruikers in staat stellen om snel samen te werken door vak verbinding te maken met Azure Active Directory (Azure AD).

![Hiermee wordt het app-logo van Box weer gegeven](./media/box.png)

> [!div class="nextstepaction"]
> [Vak try with Azure AD >](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>Wat kunt u doen met box en Azure AD?

- **Eenvoudig eenmalige aanmelding** : Elimineer meerdere gebruikers namen en wacht woorden en Vereenvoudig de toegang door gebruikers in staat te stellen zich aan te melden met hun Azure AD-accounts.
- **Gebruikers inrichten automatiseren** : Automatiseer het maken, onderhouden en verwijderen van gebruikers accounts in box wanneer gebruikers uw organisatie toevoegen, verplaatsen of verlaten.
- U kunt toegang tot Box **beveiligen, beheren en bewaken** door het Toep assen van beleid voor voorwaardelijke toegang, de toegang via toegangs beoordelingen te bepalen en de toegang tot aanmeldings logboeken te controleren.

## <a name="get-started"></a>Aan de slag

Stel [Gebruikers inrichten](https://aka.ms/BoxSSODoc) en eenmalige [aanmelding](https://aka.ms/BoxProvisioningDoc) in voor box met onze eenvoudige configuratie handleidingen.
