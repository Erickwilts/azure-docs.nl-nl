# <a name="unity-showcase-application"></a>Unity showcase applicatie

De Unity-showcasetoepassing is een op Unity gebaseerde voorbeeldtoepassing om azure remote rendering-functies aan te tonen. Het kan zowel in de Unity-editor als op Hololens 2 worden uitgevoerd.

![Showcase-app](./media/showcase-app.png "Showcase-app")

## <a name="source-repository"></a>Bronrepository

De broncode voor deze toepassing naast build/deployment-instructies is te vinden in de [ARR-samplesrepository.](https://github.com/Azure/azure-remote-rendering)
