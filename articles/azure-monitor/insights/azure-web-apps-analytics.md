# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>Analytische gegevens weergeven voor statistieken voor al uw Azure Web App-bronnen

![Web Apps-symbool](./media/azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> De Azure Web App Analytics-oplossing is verplaatst naar community-ondersteuning.  
>- De oplossing is niet langer beschikbaar via de Azure Marketplace, maar kan worden geïnstalleerd vanuit [Azure Quickstart-sjablonen](https://azure.microsoft.com/resources/templates/101-webappazure-oms-monitoring/) waar deze wordt ondersteund door de community.
>- Klanten die de oplossing al hebben geïnstalleerd, kunnen deze blijven gebruiken met wijzigingen.  
>- Microsoft raadt u aan [Application Insights](../../azure-monitor/app/app-insights-overview.md) te gebruiken om uw webtoepassingen te controleren.
