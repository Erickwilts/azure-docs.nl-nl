# <a name="storsimple-8000-series-update-51-release-notes"></a>StorSimple 8000 Series Update 5.1 release notes

## <a name="overview"></a>Overzicht

De volgende release notes beschrijven de nieuwe functies en identificeren de kritieke open problemen voor StorSimple 8000 Series Update 5.1. Ze bevatten ook een lijst van de StorSimple software-updates opgenomen in deze release.

Update 5.1 kan worden toegepast op elk StorSimple-apparaat met Update 5. Als u een versie lager dan 5 gebruikt, past u eerst update 5 toe en past u 5.1 toe. De apparaatversie die is gekoppeld aan Update 5.1 is 6.3.9600.17885.

Bekijk de informatie in de releasenotes voordat u de update implementeert in uw StorSimple-oplossing.

>
> * Update 5.1 is een verplichte update en moet onmiddellijk worden geÃ¯nstalleerd. Zie voor meer informatie hoe [u Update 5.1 toepast.](storsimple-8000-install-update-51.md)
> * Update 5.1 heeft alleen beveiligingsupdates. Het installeren van deze update duurt ongeveer 30 minuten. We raden u ten zeerste aan update 5.1 toe te passen om de werking van uw apparaat te garanderen.
> * Voor nieuwe releases ziet u mogelijk niet meteen updates omdat we een gefaseerde implementatie van de updates uitvoeren. Wacht een paar dagen en scan vervolgens opnieuw naar updates, omdat deze updates binnenkort beschikbaar zullen zijn.

## <a name="whats-new-in-update-51"></a>Nieuwe informatie in Update 5.1

De volgende belangrijke verbeteringen en bug fixes zijn gemaakt in Update 5.1:

* **TLS 1.2** - Deze StorSimple-update dwingt TLS 1.2 af op alle clients. Dit is een verplichte update voor alle StorSimple 8000-serie apparaten.

## <a name="known-issues-in-update-51-from-previous-releases"></a>Bekende problemen in Update 5.1 van eerdere releases

Er zijn geen nieuwe bekende problemen in Update 5.1. Ga naar [Update 3 release notes](storsimple-update3-release-notes.md#known-issues-in-update-3)voor een lijst met problemen die zijn overgedragen naar Update 5.1 van eerdere versies.

## <a name="storsimple-cloud-appliance-updates-in-update-51"></a>StorSimple Cloud Appliance updates in Update 5.1

Deze update kan niet worden toegepast op het StorSimple Cloud Appliance (ook wel het virtuele apparaat genoemd). Er moeten nieuwe cloudapparaten worden gemaakt met de afbeelding update 5.1. Ga voor informatie over het maken van een StorSimple Cloud Appliance naar [Implementeren en beheren van een StorSimple Cloud Appliance.](storsimple-8000-cloud-appliance-u2.md)
