# <a name="threat-modeling-tool-update-release-71607021---722019"></a>Threat Modeling Tool Update release 7.1.60702.1-7/2/2019

Versie 7.1.60702.1 van de Microsoft Threat Modeling Tool (TMT) is uitgebracht op 2 2019 juli en bevat de volgende wijzigingen:

- Verbeterde toegankelijkheid
- Opgeloste fouten
- Insluiting van een open source-community die is voorzien van een stencilset

## <a name="feature-changes"></a>Functie wijzigingen

### <a name="a-new-medical-devices-stencil-set-provided-by-the-open-source-community-is-available"></a>Er is een nieuw stencil ingesteld voor medische apparaten dat is opgegeven door de open-source community

Een stencilset voor het model leren van medische apparaten is bijgedragen door de open source-community. Na het bijwerken wordt de nieuwe stencilset weer gegeven in de vervolg keuzelijst sjabloon selectie. Lees de informatie op de [pagina github](https://github.com/Microsoft/threat-modeling-templates)van het project voor informatie over het bijdragen over stencils of inhoud naar sjablonen.

![Model validatie optie](./media/threat-modeling-tool-releases-71607021/tmt-template-selection.png)
