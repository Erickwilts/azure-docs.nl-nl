# <a name="azure-iot-hub-get-started-with-physical-devices-tutorials"></a>Azure IoT Hub aan de slag met zelfstudies voor fysieke apparaten

Deze zelfstudies laten u kennismaken met Azure IoT Hub en de sdks van het apparaat. De zelfstudies hebben betrekking op veelvoorkomende IoT-scenario's om de mogelijkheden van IoT Hub aan te tonen. De tutorials illustreren ook hoe je IoT Hub combineren met andere Azure-services en -hulpprogramma's om krachtigere IoT-oplossingen te bouwen. De zelfstudies in de volgende tabel laten zien hoe u fysieke IoT-apparaten maakt.
