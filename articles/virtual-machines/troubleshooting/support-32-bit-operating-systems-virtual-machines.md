# <a name="support-for-32-bit-operating-systems-in-azure-virtual-machines"></a>Ondersteuning voor 32-bits besturingssystemen in virtuele Azure-machines

Met Microsoft Azure kunnen gebruikers nu hun 32-bits Windows-besturingssystemen naar Azure brengen. Aangezien sommige van deze besturingssystemen al hun end-of-life supportability-overeenkomst hebben bereikt, biedt Microsoft mogelijk geen extra ondersteuning voor hen. Ondersteuning wordt ook niet aangeboden voor op Linux gebaseerde besturingssystemen (Berkeley Software Distribution), of Berkeley Software Distribution (BSD) die worden uitgevoerd op een virtuele Microsoft Azure-machine (VM).
