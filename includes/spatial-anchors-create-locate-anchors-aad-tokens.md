U kunt bijvoorbeeld Azure AD-tokens gebruiken om te integreren met Azure spatiale ankers. Als een onderneming gebruikers onderhoudt in azure AD, kunt u een Azure AD-token voor de gebruiker opgeven in de Azure spatiale-ankers SDK.
